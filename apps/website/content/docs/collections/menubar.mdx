---
title: Menubar
description: Stylextras wrapper component.
---

Wraps `aria.Menu` from `react-aria-components`, omitting `className` and `style` and adding `style?: StyleXStyles`.

## Demo
```tsx
import { MenuBar } from '@stylextras/ui/menubar';

<MenuBar />
```

## Props
```ts
import type { MenuBarProps } from '@stylextras/ui/menubar';
```
