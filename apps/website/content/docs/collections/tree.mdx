---
title: Tree
description: Stylextras wrapper component.
---

Wraps `aria.Tree` from `react-aria-components`, omitting `className` and `style` and adding `style?: StyleXStyles`.

## Demo
```tsx
import { Tree } from '@stylextras/ui/tree';

<Tree />
```

## Props
```ts
import type { TreeProps } from '@stylextras/ui/tree';
```
