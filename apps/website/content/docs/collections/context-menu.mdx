---
title: Context Menu
description: Stylextras wrapper component.
---

Wraps `aria.Menu` from `react-aria-components`, omitting `className` and `style` and adding `style?: StyleXStyles`.

## Demo
```tsx
import { ContextMenu } from '@stylextras/ui/context-menu';

<ContextMenu />
```

## Props
```ts
import type { ContextMenuProps } from '@stylextras/ui/context-menu';
```
