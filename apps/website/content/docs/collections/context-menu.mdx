---
title: Context Menu
description: Stylextras wrapper component.
---

Wraps `aria.Menu` from `react-aria-components`, omitting `className` and `style` and adding `style?: StyleXStyles`.

## Demo
```tsx
import { ContextMenu } from '@/components/stylextras';

<ContextMenu />
```

## Props
```ts
import type { ContextMenuProps } from '@/components/stylextras';
```
