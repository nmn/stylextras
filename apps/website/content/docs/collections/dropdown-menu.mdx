---
title: Dropdown Menu
description: Stylextras wrapper component.
---

Wraps `aria.Menu` from `react-aria-components`, omitting `className` and `style` and adding `style?: StyleXStyles`.

## Demo
```tsx
import { DropdownMenu } from '@/components/stylextras';

<DropdownMenu />
```

## Props
```ts
import type { DropdownMenuProps } from '@/components/stylextras';
```
