---
title: Listbox
description: Stylextras wrapper component.
---

Wraps `aria.ListBox` from `react-aria-components`, omitting `className` and `style` and adding `style?: StyleXStyles`.

## Demo
```tsx
import { ListBox } from '@/components/stylextras';

<ListBox />
```

## Props
```ts
import type { ListBoxProps } from '@/components/stylextras';
```
