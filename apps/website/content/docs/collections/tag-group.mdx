# Tag Group

Use Tag Group to present related items, actions, or records while keeping structure and styling separate from application state.

The implementation is intentionally small: visual defaults come from token variables, and the public props keep native HTML attributes available where the underlying element supports them.

## Import

```tsx
import { TagGroup } from "@stylextras/ui/tag-group";
```

## Basic usage

Start here when wiring the component into an interface. The demo shows the expected behavior, and the code block shows the smallest useful invocation.

<Suspense fallback={null}>
  <Example />
</Suspense>

```tsx
import { TagGroup } from "@stylextras/ui/tag-group";

<TagGroup>Example</TagGroup>;
```

## Props

The public prop types are exported from `@stylextras/ui/tag-group`. Most components also accept an `sx` prop for custom styles and forward safe native HTML attributes for the underlying element.

```tsx
export type TagGroupProps = Omit<BaseProps, "className" | "style"> & {
  sx?: StyleXStyles;
};
```

## Customize styles

Use the `sx` prop to compose custom styles with the component's token-driven defaults.

```tsx
import * as stylex from "@stylexjs/stylex";
import { colors } from "@stylextras/ui/tokens/color";
import { spacing } from "@stylextras/ui/tokens/spacing";
import { TagGroup } from "@stylextras/ui/tag-group";

<TagGroup sx={styles.emphasis}>Example</TagGroup>;

const styles = stylex.create({
  emphasis: {
    backgroundColor: colors.bgSubtle,
    padding: spacing.md,
  },
});
```

## Complete example

This example shows the component in a fuller composition with the surrounding pieces it usually needs.

```tsx
import { Badge } from "@stylextras/ui/badge";
import { TagGroup } from "@stylextras/ui/tag-group";

function TagGroupExample() {
  return (
    <>
      <TagGroup>
        <Badge>Design system</Badge>
        <Badge>Experimental</Badge>
        <Badge>Native popup</Badge>
      </TagGroup>
    </>
  );
}
```

## Grouped composition

Group related items when the collection should scan as a single unit.

```tsx
import { TagGroup } from "@stylextras/ui/tag-group";

<TagGroup>
  <Button>One</Button>
  <Button variant="secondary">Two</Button>
</TagGroup>;
```

## Accessibility notes

Provide meaningful item text, headings, and native roles where the surrounding product owns richer selection or keyboard behavior.
