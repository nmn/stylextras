---
title: Tag Group
description: Stylextras wrapper component.
---

Wraps `aria.TagGroup` from `react-aria-components`, omitting `className` and `style` and adding `style?: StyleXStyles`.

## Demo
```tsx
import { TagGroup } from '@/components/stylextras';

<TagGroup />
```

## Props
```ts
import type { TagGroupProps } from '@/components/stylextras';
```
