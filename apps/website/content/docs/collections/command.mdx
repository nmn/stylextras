---
title: Command
description: Stylextras wrapper component.
---

Wraps `aria.Autocomplete` from `react-aria-components`, omitting `className` and `style` and adding `style?: StyleXStyles`.

## Demo
```tsx
import { Command } from '@stylextras/ui/command';

<Command />
```

## Props
```ts
import type { CommandProps } from '@stylextras/ui/command';
```
