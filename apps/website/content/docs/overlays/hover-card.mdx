---
title: Hover Card
description: Stylextras wrapper component.
---

Wraps `aria.Popover` from `react-aria-components`, omitting `className` and `style` and adding `style?: StyleXStyles`.

## Demo
```tsx
import { HoverCard } from '@stylextras/ui/hover-card';

<HoverCard />
```

## Props
```ts
import type { HoverCardProps } from '@stylextras/ui/hover-card';
```
