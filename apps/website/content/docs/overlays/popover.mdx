`Popover` renders a native element with the `popover` attribute. It works directly with browser-level trigger attributes like `popovertarget`.

## Example

<PopoverExample />

## Notes

- Renders a native `div` with `popover="auto"` by default
- Supports both `auto` and `manual` popover behaviors
- Works with standard trigger buttons using `popovertarget`

## Props

- `size?: 'sm' | 'md' | 'lg'`
- `behavior?: 'auto' | 'manual'`
- `sx?: StyleXStyles`
