---
title: Tooltip
description: Stylextras wrapper component.
---

Wraps `aria.Tooltip` from `react-aria-components`, omitting `className` and `style` and adding `style?: StyleXStyles`.

## Demo
```tsx
import { Tooltip } from '@/components/stylextras';

<Tooltip />
```

## Props
```ts
import type { TooltipProps } from '@/components/stylextras';
```
