---
title: Drawer
description: Stylextras wrapper component.
---

Wraps `aria.Modal` from `react-aria-components`, omitting `className` and `style` and adding `style?: StyleXStyles`.

## Demo
```tsx
import { Drawer } from '@stylextras/ui/drawer';

<Drawer />
```

## Props
```ts
import type { DrawerProps } from '@stylextras/ui/drawer';
```
