---
title: Drawer
description: Stylextras wrapper component.
---

Wraps `aria.Modal` from `react-aria-components`, omitting `className` and `style` and adding `style?: StyleXStyles`.

## Demo
```tsx
import { Drawer } from '@/components/stylextras';

<Drawer />
```

## Props
```ts
import type { DrawerProps } from '@/components/stylextras';
```
