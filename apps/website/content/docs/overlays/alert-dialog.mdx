---
title: Alert Dialog
description: Stylextras wrapper component.
---

Wraps `aria.Dialog` from `react-aria-components`, omitting `className` and `style` and adding `style?: StyleXStyles`.

## Demo
```tsx
import { AlertDialog } from '@stylextras/ui/alert-dialog';

<AlertDialog />
```

## Props
```ts
import type { AlertDialogProps } from '@stylextras/ui/alert-dialog';
```
