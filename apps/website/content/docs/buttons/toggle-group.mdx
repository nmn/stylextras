---
title: Toggle Group
description: Stylextras wrapper component.
---

Wraps `aria.ToggleButtonGroup` from `react-aria-components`, omitting `className` and `style` and adding `style?: StyleXStyles`.

## Demo
```tsx
import { ToggleGroup } from '@stylextras/ui/toggle-group';

<ToggleGroup />
```

## Props
```ts
import type { ToggleGroupProps } from '@stylextras/ui/toggle-group';
```
