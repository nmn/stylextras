# Toggle Group

Use Toggle Group when an interface needs a clear, keyboard-accessible control for taking action. The component keeps native button behavior and moves visual decisions into token variables.

The implementation is intentionally small: visual defaults come from token variables, and the public props keep native HTML attributes available where the underlying element supports them.

## Import

```tsx
import { ToggleGroup } from "@stylextras/ui/toggle-group";
```

## Basic usage

Start here when wiring the component into an interface. The demo shows the expected behavior, and the code block shows the smallest useful invocation.

<Suspense fallback={null}>
  <Example />
</Suspense>

```tsx
import { ToggleGroup } from "@stylextras/ui/toggle-group";

<ToggleGroup>Example</ToggleGroup>;
```

## Props

The public prop types are exported from `@stylextras/ui/toggle-group`. Most components also accept an `sx` prop for custom styles and forward safe native HTML attributes for the underlying element.

```tsx
export type ToggleGroupProps = Omit<BaseProps, "className" | "style"> & {
  sx?: StyleXStyles;
};
```

## Customize styles

Use the `sx` prop to compose custom styles with the component's token-driven defaults.

```tsx
import * as stylex from "@stylexjs/stylex";
import { colors } from "@stylextras/ui/tokens/color";
import { spacing } from "@stylextras/ui/tokens/spacing";
import { ToggleGroup } from "@stylextras/ui/toggle-group";

<ToggleGroup sx={styles.emphasis}>Example</ToggleGroup>;

const styles = stylex.create({
  emphasis: {
    backgroundColor: colors.bgSubtle,
    padding: spacing.md,
  },
});
```

## Complete example

This example shows the component in a fuller composition with the surrounding pieces it usually needs.

```tsx
import { Toggle } from "@stylextras/ui/toggle";
import { ToggleGroup } from "@stylextras/ui/toggle-group";

function ToggleGroupExample() {
  return (
    <>
      <ToggleGroup>
        <Toggle defaultChecked>Color</Toggle>
        <Toggle defaultChecked>Spacing</Toggle>
        <Toggle>Motion</Toggle>
      </ToggleGroup>
    </>
  );
}
```

## Grouped composition

Group related items when the collection should scan as a single unit.

```tsx
import { ToggleGroup } from "@stylextras/ui/toggle-group";

<ToggleGroup>
  <Button>One</Button>
  <Button variant="secondary">Two</Button>
</ToggleGroup>;
```

## Accessibility notes

Prefer explicit visible text for actions. For icon-only controls, use the component props that require an accessible label instead of relying on the icon alone.

## Implementation notes

- Styles use token variables from `@stylextras/ui/tokens/*`.
- The implementation lives in `packages/ui/src/toggle-group/index.tsx`.
