---
title: Button Group
description: Stylextras wrapper component.
---

Wraps `aria.Toolbar` from `react-aria-components`, omitting `className` and `style` and adding `style?: StyleXStyles`.

## Demo
```tsx
import { ButtonGroup } from '@stylextras/ui/button-group';

<ButtonGroup />
```

## Props
```ts
import type { ButtonGroupProps } from '@stylextras/ui/button-group';
```
