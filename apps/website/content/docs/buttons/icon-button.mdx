---
title: Icon Button
description: Stylextras wrapper component.
---

Wraps `aria.Button` from `react-aria-components`, omitting `className` and `style` and adding `style?: StyleXStyles`.

## Demo
```tsx
import { IconButton } from '@stylextras/ui/icon-button';

<IconButton />
```

## Props
```ts
import type { IconButtonProps } from '@stylextras/ui/icon-button';
```
