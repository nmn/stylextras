---
title: Segmented Control
description: Stylextras wrapper component.
---

Wraps `aria.ToggleButtonGroup` from `react-aria-components`, omitting `className` and `style` and adding `style?: StyleXStyles`.

## Demo
```tsx
import { SegmentedControl } from '@/components/stylextras';

<SegmentedControl />
```

## Props
```ts
import type { SegmentedControlProps } from '@/components/stylextras';
```
