---
title: Typography
description: Stylextras wrapper component.
---

Wraps `aria.Text` from `react-aria-components`, omitting `className` and `style` and adding `style?: StyleXStyles`.

## Demo
```tsx
import { Typography } from '@stylextras/ui/typography';

<Typography />
```

## Props
```ts
import type { TypographyProps } from '@stylextras/ui/typography';
```
