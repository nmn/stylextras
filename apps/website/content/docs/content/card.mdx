`Card` is a thin wrapper around a native `div` with token-driven surface styling. It is useful for panels, settings groups, summaries, and other contained blocks of content.

## Example

<CardExample />

## Notes

- Renders a native `div`.
- Accepts all standard `div` props except `className` and `style`.
- Use `sx` for additional StyleX styles.
- Use `elevation` to switch between token-defined shadow levels.

## Props

- `elevation?: 'flat' | 'sm' | 'md' | 'lg'`
- `sx?: StyleXStyles`
- Native `div` props such as `id`, `role`, `aria-*`, and event handlers

## Usage

```tsx
import { Card } from '@stylextras/ui/card';
import { Typography } from '@stylextras/ui/typography';

export function Example() {
  return (
    <Card elevation="sm">
      <Typography scale="title">Project summary</Typography>
      <Typography tone="muted">
        Shared components should be simple by default and themeable from tokens.
      </Typography>
    </Card>
  );
}
```
