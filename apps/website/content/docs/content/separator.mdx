`Separator` is a minimal divider built from a styled block element. It supports horizontal and vertical orientations and can be either decorative or semantic.

## Example

<SeparatorExample />

## Notes

- Supports `horizontal` and `vertical` orientations
- Use `decorative={false}` when the separator should be announced semantically
- Use `emphasis` to choose between quieter and stronger dividers

## Props

- `orientation?: 'horizontal' | 'vertical'`
- `emphasis?: 'subtle' | 'strong'`
- `decorative?: boolean`
- `sx?: StyleXStyles`
