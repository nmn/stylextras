# Aspect Ratio

Use Aspect Ratio to shape reusable content blocks while keeping typography, spacing, and surface styling token-driven.

The implementation is intentionally small: visual defaults come from token variables, and the public props keep native HTML attributes available where the underlying element supports them.

## Import

```tsx
import { AspectRatio } from "@stylextras/ui/aspect-ratio";
```

## Basic usage

Start here when wiring the component into an interface. The demo shows the expected behavior, and the code block shows the smallest useful invocation.

<Suspense fallback={null}>
  <Example />
</Suspense>

```tsx
import { AspectRatio } from "@stylextras/ui/aspect-ratio";

<AspectRatio>Example</AspectRatio>;
```

## Props

The public prop types are exported from `@stylextras/ui/aspect-ratio`. Most components also accept an `sx` prop for custom styles and forward safe native HTML attributes for the underlying element.

```tsx
export type AspectRatioProps = Omit<BaseProps, "className" | "style"> & {
  ratio?: AspectRatioValue;
  sx?: StyleXStyles;
};
```

## Customize styles

Use the `sx` prop to compose custom styles with the component's token-driven defaults.

```tsx
import * as stylex from "@stylexjs/stylex";
import { colors } from "@stylextras/ui/tokens/color";
import { spacing } from "@stylextras/ui/tokens/spacing";
import { AspectRatio } from "@stylextras/ui/aspect-ratio";

<AspectRatio sx={styles.emphasis}>Example</AspectRatio>;

const styles = stylex.create({
  emphasis: {
    backgroundColor: colors.bgSubtle,
    padding: spacing.md,
  },
});
```
