---
title: Switch
description: Stylextras wrapper component.
---

Wraps `aria.Switch` from `react-aria-components`, omitting `className` and `style` and adding `style?: StyleXStyles`.

## Demo
```tsx
import { Switch } from '@stylextras/ui/switch';

<Switch />
```

## Props
```ts
import type { SwitchProps } from '@stylextras/ui/switch';
```
