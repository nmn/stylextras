---
title: InputGroup
description: Stylextras wrapper component.
---

Wraps `aria.Group` from `react-aria-components`, omitting `className` and `style` and adding `style?: StyleXStyles`.

## Demo
```tsx
import { InputGroup } from '@stylextras/ui/input-group';

<InputGroup />
```

## Props
```ts
import type { InputGroupProps } from '@stylextras/ui/input-group';
```
