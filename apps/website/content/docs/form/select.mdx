---
title: Select
description: Stylextras wrapper component.
---

Wraps `aria.Select` from `react-aria-components`, omitting `className` and `style` and adding `style?: StyleXStyles`.

## Demo
```tsx
import { Select } from '@/components/stylextras';

<Select />
```

## Props
```ts
import type { SelectProps } from '@/components/stylextras';
```
