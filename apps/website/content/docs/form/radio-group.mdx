---
title: Radio Group
description: Stylextras wrapper component.
---

Wraps `aria.RadioGroup` from `react-aria-components`, omitting `className` and `style` and adding `style?: StyleXStyles`.

## Demo
```tsx
import { RadioGroup } from '@/components/stylextras';

<RadioGroup />
```

## Props
```ts
import type { RadioGroupProps } from '@/components/stylextras';
```
