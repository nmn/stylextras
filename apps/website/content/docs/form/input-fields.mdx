---
title: Input Fields
description: Stylextras wrapper component.
---

Wraps `aria.Input` from `react-aria-components`, omitting `className` and `style` and adding `style?: StyleXStyles`.

## Demo
```tsx
import { Input } from '@stylextras/ui/input-fields';

<Input />
```

## Props
```ts
import type { InputProps } from '@stylextras/ui/input-fields';
```
