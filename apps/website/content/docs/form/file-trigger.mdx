---
title: File Trigger
description: Stylextras wrapper component.
---

Wraps `aria.FileTrigger` from `react-aria-components`, omitting `className` and `style` and adding `style?: StyleXStyles`.

## Demo
```tsx
import { FileTrigger } from '@/components/stylextras';

<FileTrigger />
```

## Props
```ts
import type { FileTriggerProps } from '@/components/stylextras';
```
