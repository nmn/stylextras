---
title: File Trigger
description: Stylextras wrapper component.
---

Wraps `aria.FileTrigger` from `react-aria-components`, omitting `className` and `style` and adding `style?: StyleXStyles`.

## Demo
```tsx
import { FileTrigger } from '@stylextras/ui/file-trigger';

<FileTrigger />
```

## Props
```ts
import type { FileTriggerProps } from '@stylextras/ui/file-trigger';
```
