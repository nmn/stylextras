# Text Field

Use Text Field inside forms where native input behavior, labeling, validation, and token-driven styling need to stay aligned.

The implementation is intentionally small: visual defaults come from token variables, and the public props keep native HTML attributes available where the underlying element supports them.

## Import

```tsx
import { TextField } from "@stylextras/ui/text-field";
```

## Basic usage

Start here when wiring the component into an interface. The demo shows the expected behavior, and the code block shows the smallest useful invocation.

<Suspense fallback={null}>
  <Example />
</Suspense>

```tsx
import { TextField } from "@stylextras/ui/text-field";

<TextField label="Project name" placeholder="Atlas" />;
```

## Props

The public prop types are exported from `@stylextras/ui/text-field`. Most components also accept an `sx` prop for custom styles and forward safe native HTML attributes for the underlying element.

```tsx
export type TextFieldProps = Omit<
  BaseProps,
  "aria-label" | "aria-labelledby" | "className" | "style" | "size"
> &
  AccessibleNameProps & {
    sx?: StyleXStyles;
    inputSx?: StyleXStyles;
    labelSx?: StyleXStyles;
    descriptionSx?: StyleXStyles;
    errorSx?: StyleXStyles;
    description?: ReactNode;
    error?: ReactNode;
    invalid?: boolean;
    size?: TextFieldSize;
  };
```

## Customize styles

Use the `sx` prop to compose custom styles with the component's token-driven defaults.

```tsx
import * as stylex from "@stylexjs/stylex";
import { colors } from "@stylextras/ui/tokens/color";
import { spacing } from "@stylextras/ui/tokens/spacing";
import { TextField } from "@stylextras/ui/text-field";

<TextField label="Label" sx={styles.emphasis} />;

const styles = stylex.create({
  emphasis: {
    backgroundColor: colors.bgSubtle,
    padding: spacing.md,
  },
});
```

## Complete example

This example shows the component in a fuller composition with the surrounding pieces it usually needs.

```tsx
import { TextField } from "@stylextras/ui/text-field";

function TextFieldExample() {
  return (
    <>
      <div>
        <TextField label="Name" placeholder="Workspace name" />
        <TextField label="Slug" defaultValue="workspace-name" />
        <TextField disabled label="Disabled" defaultValue="Disabled value" />
      </div>
    </>
  );
}
```

## Accessibility notes

Every interactive form control should have a label or accessible name. Preserve native attributes such as required, disabled, min, max, and aria-describedby when they apply.

## Implementation notes

- Styles use token variables from `@stylextras/ui/tokens/*`.
- The implementation lives in `packages/ui/src/text-field/index.tsx`.
