---
title: Text Field
description: Stylextras wrapper component.
---

Wraps `aria.TextField` from `react-aria-components`, omitting `className` and `style` and adding `style?: StyleXStyles`.

## Demo
```tsx
import { TextField } from '@stylextras/ui/text-field';

<TextField />
```

## Props
```ts
import type { TextFieldProps } from '@stylextras/ui/text-field';
```
