---
title: Checkbox
description: Stylextras wrapper component.
---

Wraps `aria.Checkbox` from `react-aria-components`, omitting `className` and `style` and adding `style?: StyleXStyles`.

## Demo
```tsx
import { Checkbox } from '@stylextras/ui/checkbox';

<Checkbox />
```

## Props
```ts
import type { CheckboxProps } from '@stylextras/ui/checkbox';
```
