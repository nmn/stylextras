---
title: Number Field
description: Stylextras wrapper component.
---

Wraps `aria.NumberField` from `react-aria-components`, omitting `className` and `style` and adding `style?: StyleXStyles`.

## Demo
```tsx
import { NumberField } from '@stylextras/ui/number-field';

<NumberField />
```

## Props
```ts
import type { NumberFieldProps } from '@stylextras/ui/number-field';
```
