`Slider` is a thin wrapper around a native `input type="range"` with token-driven sizing and label styling.

## Example

<SliderExample />

## Notes

- Renders a native `input` with `type="range"`
- Uses `accent-color` for a simple standards-based appearance
- Accepts all standard range input props except `className`, `style`, and `type`

## Props

- `label?: ReactNode`
- `size?: 'sm' | 'md'`
- `sx?: StyleXStyles`
- `inputSx?: StyleXStyles`

```ts
import { Slider } from '@stylextras/ui/slider';
import type { SliderProps } from '@stylextras/ui/slider';

export function Example() {
  return <Slider label="Volume" defaultValue={40} max={100} min={0} />;
}
```
