---
title: Slider
description: Stylextras wrapper component.
---

Wraps `aria.Slider` from `react-aria-components`, omitting `className` and `style` and adding `style?: StyleXStyles`.

## Demo
```tsx
import { Slider } from '@/components/stylextras';

<Slider />
```

## Props
```ts
import type { SliderProps } from '@/components/stylextras';
```
