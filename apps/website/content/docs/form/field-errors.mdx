---
title: FieldErrors
description: Stylextras wrapper component.
---

Wraps `aria.FieldError` from `react-aria-components`, omitting `className` and `style` and adding `style?: StyleXStyles`.

## Demo
```tsx
import { FieldErrors } from '@/components/stylextras';

<FieldErrors />
```

## Props
```ts
import type { FieldErrorsProps } from '@/components/stylextras';
```
