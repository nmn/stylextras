---
title: FieldErrors
description: Stylextras wrapper component.
---

Wraps `aria.FieldError` from `react-aria-components`, omitting `className` and `style` and adding `style?: StyleXStyles`.

## Demo
```tsx
import { FieldErrors } from '@stylextras/ui/field-errors';

<FieldErrors />
```

## Props
```ts
import type { FieldErrorsProps } from '@stylextras/ui/field-errors';
```
