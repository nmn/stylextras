---
title: Textarea
description: Stylextras wrapper component.
---

Wraps `aria.TextArea` from `react-aria-components`, omitting `className` and `style` and adding `style?: StyleXStyles`.

## Demo
```tsx
import { TextArea } from '@/components/stylextras';

<TextArea />
```

## Props
```ts
import type { TextAreaProps } from '@/components/stylextras';
```
