# Combo Box

Use Combo Box inside forms where native input behavior, labeling, validation, and token-driven styling need to stay aligned.

The implementation is intentionally small: visual defaults come from token variables, and the public props keep native HTML attributes available where the underlying element supports them.

## Import

```tsx
import { ComboBoxField, ComboBox } from "@stylextras/ui/combo-box";
```

## Basic usage

Start here when wiring the component into an interface. The demo shows the expected behavior, and the code block shows the smallest useful invocation.

<Suspense fallback={null}>
  <Example />
</Suspense>

```tsx
import { ComboBoxField, ComboBox } from "@stylextras/ui/combo-box";

<ComboBoxField>Example</ComboBoxField>;
```

## Props

The public prop types are exported from `@stylextras/ui/combo-box`. Most components also accept an `sx` prop for custom styles and forward safe native HTML attributes for the underlying element.

```tsx
export type ComboBoxProps = ComboBoxProps;

export type ComboBoxFieldProps = ComboBoxProps;
```

## Complete example

This example shows the component in a fuller composition with the surrounding pieces it usually needs.

```tsx
import { ComboBox } from "@stylextras/ui/combo-box";

function ComboBoxExample() {
  return (
    <>
      <div>
        <ComboBox
          label="Reviewer"
          listId="reviewers"
          options={["Alex Kim", "Taylor Rivera", "Jordan Patel", "Sam Lee"]}
        />
        <ComboBox
          label="Environment"
          listId="environments"
          options={["Development", "Preview", "Production"]}
        />
      </div>
    </>
  );
}
```

## Basic field

This pattern shows another common way to compose the component.

```tsx
import { ComboBox } from "@stylextras/ui/combo-box";

<ComboBox label="Label" />;
```

## With native props

This pattern shows another common way to compose the component.

```tsx
<ComboBox label="Required field" required disabled={false} />
```

## Accessibility notes

Every interactive form control should have a label or accessible name. Preserve native attributes such as required, disabled, min, max, and aria-describedby when they apply.

## Implementation notes

- Styles use token variables from `@stylextras/ui/tokens/*`.
- The implementation lives in `packages/ui/src/combo-box/index.tsx`.
