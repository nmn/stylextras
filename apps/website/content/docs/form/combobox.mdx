---
title: Combobox
description: Stylextras wrapper component.
---

Wraps `aria.ComboBox` from `react-aria-components`, omitting `className` and `style` and adding `style?: StyleXStyles`.

## Demo
```tsx
import { ComboBox } from '@/components/stylextras';

<ComboBox />
```

## Props
```ts
import type { ComboBoxProps } from '@/components/stylextras';
```
