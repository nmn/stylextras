# Combobox

Use Combobox inside forms where native input behavior, labeling, validation, and token-driven styling need to stay aligned.

The implementation is intentionally small: visual defaults come from token variables, and the public props keep native HTML attributes available where the underlying element supports them.

## Import

```tsx
import { Combobox } from "@stylextras/ui/combobox";
```

## Basic usage

Start here when wiring the component into an interface. The demo shows the expected behavior, and the code block shows the smallest useful invocation.

<Suspense fallback={null}>
  <Example />
</Suspense>

```tsx
import { Combobox } from "@stylextras/ui/combobox";

<Combobox>Example</Combobox>;
```

## Props

The public prop types are exported from `@stylextras/ui/combobox`. Most components also accept an `sx` prop for custom styles and forward safe native HTML attributes for the underlying element.

```tsx
export type ComboboxProps = Omit<BaseProps, "className" | "style" | "type"> & {
  sx?: StyleXStyles;
  inputSx?: StyleXStyles;
  labelSx?: StyleXStyles;
  label?: ReactNode;
  listId?: string;
  options?: ComboboxOption[];
};
```

## Customize styles

Use the `sx` prop to compose custom styles with the component's token-driven defaults.

```tsx
import * as stylex from "@stylexjs/stylex";
import { colors } from "@stylextras/ui/tokens/color";
import { spacing } from "@stylextras/ui/tokens/spacing";
import { Combobox } from "@stylextras/ui/combobox";

<Combobox label="Label" sx={styles.emphasis} />;

const styles = stylex.create({
  emphasis: {
    backgroundColor: colors.bgSubtle,
    padding: spacing.md,
  },
});
```

## Complete example

This example shows the component in a fuller composition with the surrounding pieces it usually needs.

```tsx
import { Combobox } from "@stylextras/ui/combobox";

function ComboboxExample() {
  return (
    <>
      <div>
        <Combobox
          label="Reviewer"
          listId="reviewers"
          options={["Alex Kim", "Taylor Rivera", "Jordan Patel", "Sam Lee"]}
        />
        <Combobox
          label="Environment"
          listId="environments"
          options={["Development", "Preview", "Production"]}
        />
      </div>
    </>
  );
}
```

This pattern shows another common way to compose the component.


## Accessibility notes

Every interactive form control should have a label or accessible name. Preserve native attributes such as required, disabled, min, max, and aria-describedby when they apply.

## Implementation notes

- Styles use token variables from `@stylextras/ui/tokens/*`.
- The implementation lives in `packages/ui/src/combobox/index.tsx`.
