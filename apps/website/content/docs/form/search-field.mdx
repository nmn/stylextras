---
title: Search Field
description: Stylextras wrapper component.
---

Wraps `aria.SearchField` from `react-aria-components`, omitting `className` and `style` and adding `style?: StyleXStyles`.

## Demo
```tsx
import { SearchField } from '@stylextras/ui/search-field';

<SearchField />
```

## Props
```ts
import type { SearchFieldProps } from '@stylextras/ui/search-field';
```
