## Anatomy

- `ColorPicker` — Native `<input type="color">`.

## Accessibility and browser behavior

`ColorPicker` is a zero-JavaScript native color input with a wider preview. The browser supplies its picker, keyboard behavior, validation, and form participation. Give it a visible label or an accessible name.

Any native input prop not named in `Omit` remains accepted, including `name`, `form`, standard ARIA/data attributes, event handlers, and `ref`. `sx` is composed last.

## API reference

### ColorPicker

`ColorPickerProps`

Complete native/inherited prop base: `ColorFieldProps`, which is `Omit<React.ComponentPropsWithRef<"input">, "className" | "style" | "type">`. The props below complete `ColorPickerProps`.

- `sx` — `stylex.StyleXStyles`.
