## Accessibility and browser behavior

`TimeField` is a zero-JavaScript native time input. The browser supplies localized editing, keyboard behavior, validation, and form participation. Values for `defaultValue`, `value`, `min`, and `max` use the 24-hour `HH:mm` format even when the browser presents a localized 12-hour interface.

Give the input a visible label. The component adds a visible focus outline, invalid-state border, forced-colors fallback, minimum target, and reduced-motion transition behavior. Any native input prop not named in `Omit` remains accepted, including standard ARIA/data attributes, event handlers, and `ref`. `TimeFieldField` remains as a deprecated compatibility alias. `sx` is composed last.
