## Anatomy

- `ColorWheel` — Native `<input type="color">`.

## Accessibility and browser behavior

`ColorWheel` is a circular presentation of the native color input, not a custom polar control. The browser supplies its picker, keyboard behavior, validation, and form participation without component JavaScript. Give it a visible label or an accessible name.

Any native input prop not named in `Omit` remains accepted, including standard ARIA/data attributes, event handlers, and `ref`. `sx` is composed last.

## API reference

### ColorWheel

`ColorWheelProps`

Complete native/inherited prop base: `ColorFieldProps`, which is `Omit<React.ComponentPropsWithRef<"input">, "className" | "style" | "type">`. The props below complete `ColorWheelProps`.

- `sx` — `stylex.StyleXStyles`.
