## Accessibility and browser behavior

`ColorSwatch` is decorative and receives `aria-hidden="true"` by default. Supplying `aria-label` or `aria-labelledby` makes it a named `role="img"`. A named swatch should describe the color in words, not only repeat a hex value.

The visible border keeps very light and very dark colors distinguishable from their surroundings. The component uses no JavaScript. Native span props, including standard ARIA/data attributes and `ref`, remain accepted. `sx` is composed last.

## API reference

### ColorSwatch

`ColorSwatchProps`

Complete native/inherited prop base: `Omit<React.ComponentPropsWithRef<"span">, "aria-hidden" | "aria-label" | "aria-labelledby" | "className" | "role" | "style">`, combined with either a required accessible name or decorative `aria-hidden`. The props below complete `ColorSwatchProps`.

- `color` — `string`. Default: `"#7c3aed"`.
- `sx` — `stylex.StyleXStyles`.
