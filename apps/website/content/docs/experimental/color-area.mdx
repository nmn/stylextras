## Anatomy

## Accessibility and browser behavior

`ColorArea` is a zero-JavaScript native color input. The browser supplies its picker, keyboard behavior, validation, and form participation. The area proportions are presentation only; use `ColorField` when those proportions would imply unsupported two-dimensional interaction.

Give the input a visible label or an accessible name. Any native input prop not named in `Omit` remains accepted, including `name`, `form`, `disabled`, standard ARIA/data attributes, event handlers, and `ref`. `sx` is composed last.

## API reference

### ColorArea

`ColorAreaProps`

Complete native/inherited prop base: `ColorFieldProps`, which is `Omit<React.ComponentPropsWithRef<"input">, "className" | "style" | "type">`. The props below complete `ColorAreaProps`.

- `sx` — `stylex.StyleXStyles`.
