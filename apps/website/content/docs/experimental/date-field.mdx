## Anatomy

- `DateField` — Native `<input type="date">`.

## Accessibility and browser behavior

`DateField` is an experimental compatibility name for `DatePicker`; both render a zero-JavaScript native date input. The browser supplies localized editing, validation, keyboard behavior, and form participation. Values for `defaultValue`, `value`, `min`, and `max` use the ISO `YYYY-MM-DD` format.

Give the input a visible label. Any native input prop not named in `Omit` remains accepted, including standard ARIA/data attributes, event handlers, and `ref`. `sx` is composed last.

## API reference

### DateField

`DateFieldProps`

Complete native/inherited prop base: `DatePickerProps`, which is `Omit<React.ComponentPropsWithRef<"input">, "className" | "style" | "type">` plus the deprecated `inputId` alias and `sx`. The props below complete `DateFieldProps`.

- `sx` — `stylex.StyleXStyles`.
