## Anatomy

- `TagList` — `<ul role="list">` that wraps each child in an `<li>`.

## Accessibility and browser behavior

`TagList` preserves native list semantics without JavaScript. It is intended for static metadata tags. Use native checkbox or radio controls when tags are selectable, and buttons when each tag performs an action.

Native `ul` props, including accessible naming, standard ARIA/data attributes, event handlers, and `ref`, remain accepted. `TagGroup` remains as a deprecated compatibility alias. `sx` is composed last.

## API reference

### TagList

`TagListProps`

Complete native/inherited prop base: `Omit<React.ComponentPropsWithRef<"ul">, "className" | "style">`. The props below complete `TagListProps`.

- `sx` — `stylex.StyleXStyles`.

### TagGroup (deprecated alias)

`TagGroupProps`

`TagGroup` aliases `TagList`, and `TagGroupProps` is the same type as `TagListProps`. Prefer the list name because these tags are static metadata, not interactive group controls.
