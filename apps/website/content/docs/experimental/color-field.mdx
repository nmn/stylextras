## Anatomy

- `ColorField` — Native `<input type="color">`.

## Accessibility and browser behavior

`ColorField` is a zero-JavaScript native color input. The browser supplies its picker, keyboard behavior, validation, and form participation. Give the input a visible label or an accessible name.

The component adds a visible focus outline, invalid-state border, forced-colors border, disabled treatment, and minimum pointer target. Any native input prop not named in `Omit` remains accepted, including standard ARIA/data attributes, event handlers, and `ref`. `sx` is composed last.
