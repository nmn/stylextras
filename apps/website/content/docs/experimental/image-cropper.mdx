## Anatomy

- `ImageCropPreview` — Fixed-ratio `<div>` frame containing an `<img>`.

## Accessibility and browser behavior

`ImageCropPreview` is a visual preview, not an interactive crop editor. It uses CSS `aspect-ratio`, `object-fit`, logical RTL-aware positions, and local overflow containment without JavaScript. Supply `alt=""` for a decorative image or meaningful alternative text when the image conveys content; `alt` is required by the type.

Native image props not named in `Omit` remain accepted. `frameSx` styles the containing frame, and `sx` styles the image. `ImageCropper` remains as a deprecated compatibility alias.

## API reference

### ImageCropPreview

`ImageCropPreviewProps`

Complete native/inherited prop base: `Omit<React.ComponentPropsWithoutRef<"img">, "alt" | "className" | "style">`. The props below complete `ImageCropPreviewProps`.

- `alt` — `string`. Required.
- `frameSx` — `stylex.StyleXStyles`.
- `position` — `ImageCropperPosition`. Default: `'center'`.
- `ratio` — `ImageCropperRatio`. Default: `'square'`.
- `sx` — `stylex.StyleXStyles`.

### ImageCropper (deprecated alias)

`ImageCropperProps`

`ImageCropper` aliases `ImageCropPreview`, and `ImageCropperProps` is the same type as `ImageCropPreviewProps`. Prefer the preview name because this CSS-only component does not edit or export a crop.
