# Alert Dialog

Use Alert Dialog for portal-free popup interactions built from trigger and content components. These docs show lazy content loading and direct content composition.

The implementation is intentionally small: visual defaults come from token variables, and the public props keep native HTML attributes available where the underlying element supports them.

## Import

```tsx
import {
  AlertDialogContent,
  AlertDialogTrigger,
  AlertDialogProgrammatic,
} from "@stylextras/ui/alert-dialog";
```

## Basic usage

Start here when wiring the component into an interface. The demo shows the expected behavior, and the code block shows the smallest useful invocation.

<Suspense fallback={null}>
  <Example />
</Suspense>

```tsx
import {
  AlertDialogContent,
  AlertDialogTrigger,
  AlertDialogProgrammatic,
} from "@stylextras/ui/alert-dialog";

function AlertDialogExampleContent(props: AlertDialogContentProps) {
  return <AlertDialogContent {...props}>Details</AlertDialogContent>;
}

<AlertDialogTrigger content={() => Promise.resolve(AlertDialogExampleContent)}>
  Open Alert Dialog
</AlertDialogTrigger>;
```

## Props

The public prop types are exported from `@stylextras/ui/alert-dialog`. Most components also accept an `sx` prop for custom styles and forward safe native HTML attributes for the underlying element.

```tsx
export type AlertDialogProps = Omit<BaseProps, "className" | "style"> & {
  sx?: StyleXStyles;
};

export type AlertDialogContentProps = AlertDialogProps;

export type AlertDialogTriggerProps = Omit<
  ButtonProps,
  "className" | "content" | "style"
> & {
  content: LazyComponentLoader<AlertDialogContentProps>;
  contentProps?: Omit<AlertDialogContentProps, "ref">;
  fallback?: React.ReactNode;
};

export type AlertDialogProgrammaticProps =
  DialogProgrammaticProps<AlertDialogContentProps>;
```

## Customize styles

Use the `sx` prop to compose custom styles with the component's token-driven defaults.

```tsx
import * as stylex from "@stylexjs/stylex";
import { colors } from "@stylextras/ui/tokens/color";
import { spacing } from "@stylextras/ui/tokens/spacing";
import {
  AlertDialogContent,
  AlertDialogTrigger,
  AlertDialogProgrammatic,
} from "@stylextras/ui/alert-dialog";

<AlertDialogContent sx={styles.emphasis}>Example</AlertDialogContent>;

const styles = stylex.create({
  emphasis: {
    backgroundColor: colors.bgSubtle,
    padding: spacing.md,
  },
});
```

## Complete example

This example shows the component in a fuller composition with the surrounding pieces it usually needs.

```tsx
import { Button } from "@stylextras/ui/button";
import { ButtonGroupActions } from "@stylextras/ui/button-group";
import { Typography } from "@stylextras/ui/typography";
import {
  AlertDialogContent,
  AlertDialogTrigger,
} from "@stylextras/ui/alert-dialog";

function AlertDialogExampleContent(props: AlertDialogContentProps) {
  return (
    <AlertDialogContent {...props}>
      <div>
        <Typography as="h3" scale="title">
          Delete draft?
        </Typography>
        <span>This action cannot be undone.</span>
        <form method="dialog">
          <ButtonGroupActions
            secondary={
              <Button variant="secondary" type="submit">
                Cancel
              </Button>
            }
            primary={
              <Button variant="danger" type="submit">
                Delete
              </Button>
            }
          />
        </form>
      </div>
    </AlertDialogContent>
  );
}

function AlertDialogExample() {
  return (
    <AlertDialogTrigger
      content={() => Promise.resolve(AlertDialogExampleContent)}
      variant="primary"
    >
      Open Dialog
    </AlertDialogTrigger>
  );
}
```

## Lazy trigger and content

Pass a content loader to the trigger so the popup body can be prefetched and rendered only when needed.

```tsx
import {
  AlertDialogTrigger,
  AlertDialogContent,
} from "@stylextras/ui/alert-dialog";

function DetailsContent(props: AlertDialogContentProps) {
  return (
    <AlertDialogContent {...props}>
      <h2>Details</h2>
      <p>Render the content only after the trigger is activated.</p>
    </AlertDialogContent>
  );
}

<AlertDialogTrigger content={() => Promise.resolve(DetailsContent)}>
  Open Alert Dialog
</AlertDialogTrigger>;
```

## Content rendered directly

Render the content component directly when another controller owns the open state.

```tsx
import { AlertDialogContent } from "@stylextras/ui/alert-dialog";

<AlertDialogContent>
  Content can still be rendered directly when another controller owns opening.
</AlertDialogContent>;
```

## Programmatic control

Use the programmatic component when open state belongs to application state rather than a trigger.

```tsx
import {
  AlertDialogProgrammatic,
  AlertDialogContent,
  type AlertDialogContentProps,
} from "@stylextras/ui/alert-dialog";

function ProgrammaticExample({ open, onOpenChange }) {
  return (
    <AlertDialogProgrammatic
      open={open}
      onOpenChange={onOpenChange}
      content={() =>
        Promise.resolve((props: AlertDialogContentProps) => (
          <AlertDialogContent {...props}>Controlled content</AlertDialogContent>
        ))
      }
    />
  );
}
```

## Accessibility notes

Keep trigger text specific, label the popup content when it has a title, and prefer the platform dialog or popover behavior before adding custom focus management.

## Implementation notes

- Styles use token variables from `@stylextras/ui/tokens/*`.
- The implementation lives in `packages/ui/src/alert-dialog/index.tsx`.
