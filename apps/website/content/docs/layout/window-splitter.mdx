# Window Splitter

Use Window Splitter to compose page structure without leaking layout-specific CSS into feature components.

The implementation is intentionally small: visual defaults come from token variables, and the public props keep native HTML attributes available where the underlying element supports them.

## Import

```tsx
import { WindowSplitter } from "@stylextras/ui/window-splitter";
```

## Basic usage

Use Window Splitter when two panes need an accessible draggable separator. Drag the splitter, or focus it and use arrow keys.

<Suspense fallback={null}>

</Suspense>

```tsx
import { WindowSplitter } from "@stylextras/ui/window-splitter";

<WindowSplitter
  label="Resize navigator"
  onValueChange={setSplit}
  value={split}
/>;
```

## Props

The public prop types are exported from `@stylextras/ui/window-splitter`. Most components also accept an `sx` prop for custom styles and forward safe native HTML attributes for the underlying element.

```tsx
export type WindowSplitterProps = Omit<BaseProps, "className" | "style"> & {
  defaultValue?: number;
  label: string;
  max?: number;
  min?: number;
  onValueChange?: (value: number) => void;
  orientation?: WindowSplitterOrientation;
  sx?: StyleXStyles;
  value?: number;
};
```

## Customize styles

Use the `sx` prop to compose custom styles with the component's token-driven defaults.

```tsx
import * as stylex from "@stylexjs/stylex";
import { colors } from "@stylextras/ui/tokens/color";
import { spacing } from "@stylextras/ui/tokens/spacing";
import { WindowSplitter } from "@stylextras/ui/window-splitter";

<WindowSplitter label="Resize panels" sx={styles.emphasis} />;

const styles = stylex.create({
  emphasis: {
    backgroundColor: colors.bgSubtle,
    padding: spacing.md,
  },
});
```

## Complete example

This example shows a controlled splitter driving pane sizes.

```tsx
import { useState } from "react";
import { WindowSplitter } from "@stylextras/ui/window-splitter";

function WindowSplitterExample() {
  const [split, setSplit] = useState(38);

  return (
    <div style={{ display: "grid", gridTemplateColumns: `${split}% auto 1fr` }}>
      <aside>Navigator</aside>
      <WindowSplitter
        label="Resize navigator"
        onValueChange={setSplit}
        value={split}
      />
      <main>Editor</main>
    </div>
  );
}
```

## Accessibility notes

Window Splitter renders a focusable `separator` with `aria-valuenow`, `aria-valuemin`, and `aria-valuemax`. Use a specific `label`, keep semantic panes around it, and support arrow-key resizing. Arrow keys move by 2%, Shift+Arrow moves by 10%, Home moves to `min`, and End moves to `max`.

## Implementation notes

- Styles use token variables from `@stylextras/ui/tokens/*`.
- The implementation lives in `packages/ui/src/window-splitter/index.tsx`.
