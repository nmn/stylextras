---
title: WindowSplitter
description: Stylextras wrapper component.
---

Wraps `aria.Separator` from `react-aria-components`, omitting `className` and `style` and adding `style?: StyleXStyles`.

## Demo
```tsx
import { WindowSplitter } from '@/components/stylextras';

<WindowSplitter />
```

## Props
```ts
import type { WindowSplitterProps } from '@/components/stylextras';
```
