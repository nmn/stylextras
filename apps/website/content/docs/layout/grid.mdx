---
title: Grid
description: Stylextras wrapper component.
---

Wraps `aria.GridList` from `react-aria-components`, omitting `className` and `style` and adding `style?: StyleXStyles`.

## Demo
```tsx
import { Grid } from '@/components/stylextras';

<Grid />
```

## Props
```ts
import type { GridProps } from '@/components/stylextras';
```
