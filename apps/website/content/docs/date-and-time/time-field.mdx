# Time Field

Use Time Field for date or time entry flows that can lean on native form controls and browser validation.

The implementation is intentionally small: visual defaults come from token variables, and the public props keep native HTML attributes available where the underlying element supports them.

Search aliases: timefield, time input, clock input.

## Import

```tsx
import { TimeFieldField, TimeField } from "@stylextras/ui/time-field";
```

## Basic usage

Start here when wiring the component into an interface. The demo shows the expected behavior, and the code block shows the smallest useful invocation.

<Suspense fallback={null}>
  <Example />
</Suspense>

```tsx
import { TimeFieldField, TimeField } from "@stylextras/ui/time-field";

<TimeField aria-label="Start time" defaultValue="09:30" />;
```

## Props

The public prop types are exported from `@stylextras/ui/time-field`. Most components also accept an `sx` prop for custom styles and forward safe native HTML attributes for the underlying element.

```tsx
export type TimeFieldProps = TimeFieldProps;

export type TimeFieldFieldProps = TimeFieldProps;
```

## Complete example

This example shows the component in a fuller composition with the surrounding pieces it usually needs.

```tsx
import { TimeField } from "@stylextras/ui/time-field";

function TimeFieldExample() {
  return (
    <>
      <div>
        <TimeField aria-label="Start time" defaultValue="09:30" />
        <TimeField aria-label="End time" defaultValue="17:30" />
      </div>
    </>
  );
}
```
