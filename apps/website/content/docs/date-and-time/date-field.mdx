---
title: Date Field
description: Stylextras wrapper component.
---

Wraps `aria.DateField` from `react-aria-components`, omitting `className` and `style` and adding `style?: StyleXStyles`.

## Demo
```tsx
import { DateField } from '@stylextras/ui/date-field';

<DateField />
```

## Props
```ts
import type { DateFieldProps } from '@stylextras/ui/date-field';
```
