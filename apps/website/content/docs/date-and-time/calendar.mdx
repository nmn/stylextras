---
title: Calendar
description: Stylextras wrapper component.
---

Wraps `aria.Calendar` from `react-aria-components`, omitting `className` and `style` and adding `style?: StyleXStyles`.

## Demo
```tsx
import { Calendar } from '@/components/stylextras';

<Calendar />
```

## Props
```ts
import type { CalendarProps } from '@/components/stylextras';
```
