# Date Range Picker

Use Date Range Picker for date or time entry flows that can lean on native form controls and browser validation.

The implementation is intentionally small: visual defaults come from token variables, and the public props keep native HTML attributes available where the underlying element supports them.

## Import

```tsx
import { DateRangePicker } from "@stylextras/ui/date-range-picker";
```

## Basic usage

Start here when wiring the component into an interface. The demo shows the expected behavior, and the code block shows the smallest useful invocation.

<Suspense fallback={null}>
  <Example />
</Suspense>

```tsx
import { DateRangePicker } from "@stylextras/ui/date-range-picker";

<DateRangePicker>Example</DateRangePicker>;
```

## Props

The public prop types are exported from `@stylextras/ui/date-range-picker`. Most components also accept an `sx` prop for custom styles and forward safe native HTML attributes for the underlying element.

```tsx
export type DateRangePickerProps = DateRangePickerProps;
```

## Labeled range control

This pattern shows another common way to compose the component.

```tsx

<DateRangePicker aria-label="Project schedule" />;
```

## Accessibility notes

Provide an accessible label and keep helper or validation text close to the field that it describes.
