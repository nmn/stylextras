---
title: Range Calendar
description: Stylextras wrapper component.
---

Wraps `aria.RangeCalendar` from `react-aria-components`, omitting `className` and `style` and adding `style?: StyleXStyles`.

## Demo
```tsx
import { RangeCalendar } from '@/components/stylextras';

<RangeCalendar />
```

## Props
```ts
import type { RangeCalendarProps } from '@/components/stylextras';
```
