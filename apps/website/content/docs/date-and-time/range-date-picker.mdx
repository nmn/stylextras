---
title: RangeDatePicker
description: Stylextras wrapper component.
---

Wraps `aria.DateRangePicker` from `react-aria-components`, omitting `className` and `style` and adding `style?: StyleXStyles`.

## Demo
```tsx
import { RangeDatePicker } from '@stylextras/ui/range-date-picker';

<RangeDatePicker />
```

## Props
```ts
import type { RangeDatePickerProps } from '@stylextras/ui/range-date-picker';
```
