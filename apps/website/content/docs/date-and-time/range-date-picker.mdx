# Range Date Picker

Use Range Date Picker for date or time entry flows that can lean on native form controls and browser validation.

The implementation is intentionally small: visual defaults come from token variables, and the public props keep native HTML attributes available where the underlying element supports them.

## Import

```tsx
import { RangeDatePicker } from "@stylextras/ui/range-date-picker";
```

## Basic usage

Start here when wiring the component into an interface. The demo shows the expected behavior, and the code block shows the smallest useful invocation.

<Suspense fallback={null}>
  <Example />
</Suspense>

```tsx
import { RangeDatePicker } from "@stylextras/ui/range-date-picker";

<RangeDatePicker>Example</RangeDatePicker>;
```

## Props

The public prop types are exported from `@stylextras/ui/range-date-picker`. Most components also accept an `sx` prop for custom styles and forward safe native HTML attributes for the underlying element.

```tsx
export type RangeDatePickerProps = Omit<BaseProps, "className" | "style"> & {
  sx?: StyleXStyles;
};
```

## Customize styles

Use the `sx` prop to compose custom styles with the component's token-driven defaults.

```tsx
import * as stylex from "@stylexjs/stylex";
import { colors } from "@stylextras/ui/tokens/color";
import { spacing } from "@stylextras/ui/tokens/spacing";
import { RangeDatePicker } from "@stylextras/ui/range-date-picker";

<RangeDatePicker sx={styles.emphasis}>Example</RangeDatePicker>;

const styles = stylex.create({
  emphasis: {
    backgroundColor: colors.bgSubtle,
    padding: spacing.md,
  },
});
```
