---
title: DatePicker
description: Stylextras wrapper component.
---

Wraps `aria.DatePicker` from `react-aria-components`, omitting `className` and `style` and adding `style?: StyleXStyles`.

## Demo
```tsx
import { DatePicker } from '@stylextras/ui/date-picker';

<DatePicker />
```

## Props
```ts
import type { DatePickerProps } from '@stylextras/ui/date-picker';
```
