# Date Picker

Use Date Picker for date or time entry flows that can lean on native form controls and browser validation.

The implementation is intentionally small: visual defaults come from token variables, and the public props keep native HTML attributes available where the underlying element supports them.

## Import

```tsx
import { DatePicker } from "@stylextras/ui/date-picker";
```

## Basic usage

Start here when wiring the component into an interface. The demo shows the expected behavior, and the code block shows the smallest useful invocation.

<Suspense fallback={null}>
  <Example />
</Suspense>

```tsx
import { DatePicker } from "@stylextras/ui/date-picker";

<DatePicker>Example</DatePicker>;
```

## Props

The public prop types are exported from `@stylextras/ui/date-picker`. Most components also accept an `sx` prop for custom styles and forward safe native HTML attributes for the underlying element.

```tsx
export type DatePickerProps = Omit<
  BaseProps,
  "className" | "style" | "type"
> & { sx?: StyleXStyles };
```

## Customize styles

Use the `sx` prop to compose custom styles with the component's token-driven defaults.

```tsx
import * as stylex from "@stylexjs/stylex";
import { colors } from "@stylextras/ui/tokens/color";
import { spacing } from "@stylextras/ui/tokens/spacing";
import { DatePicker } from "@stylextras/ui/date-picker";

<DatePicker sx={styles.emphasis}>Example</DatePicker>;

const styles = stylex.create({
  emphasis: {
    backgroundColor: colors.bgSubtle,
    padding: spacing.md,
  },
});
```

## Complete example

This example shows the component in a fuller composition with the surrounding pieces it usually needs.

```tsx
import { DatePicker } from "@stylextras/ui/date-picker";

function DatePickerExample() {
  return (
    <>
      <div>
        <DatePicker aria-label="Publish date" defaultValue="2026-04-12" />
        <DatePicker aria-label="Archive date" defaultValue="2026-04-20" />
      </div>
    </>
  );
}
```

Provide an accessible label and keep helper or validation text close to the field that it describes.
