# Progress Bar

Use Progress Bar to communicate state, progress, or feedback in a compact token-driven surface.

The implementation is intentionally small: visual defaults come from token variables, and the public props keep native HTML attributes available where the underlying element supports them.

## Import

```tsx
import { ProgressBar } from "@stylextras/ui/progress-bar";
```

## Basic usage

Start here when wiring the component into an interface. The demo shows the expected behavior, and the code block shows the smallest useful invocation.

<Suspense fallback={null}>
  <Example />
</Suspense>

```tsx
import { ProgressBar } from "@stylextras/ui/progress-bar";

<ProgressBar aria-label="Upload progress" value={64} max={100} />;
```

## Props

The public prop types are exported from `@stylextras/ui/progress-bar`. Most components also accept an `sx` prop for custom styles and forward safe native HTML attributes for the underlying element.

```tsx
export type ProgressBarProps = Omit<
  BaseProps,
  "aria-label" | "aria-labelledby" | "className" | "style"
> &
  AccessibleNameProps & {
    sx?: StyleXStyles;
  };
```

## Customize styles

Use the `sx` prop to compose custom styles with the component's token-driven defaults.

```tsx
import * as stylex from "@stylexjs/stylex";
import { colors } from "@stylextras/ui/tokens/color";
import { spacing } from "@stylextras/ui/tokens/spacing";
import { ProgressBar } from "@stylextras/ui/progress-bar";

<ProgressBar label="Label" sx={styles.emphasis} />;

const styles = stylex.create({
  emphasis: {
    backgroundColor: colors.bgSubtle,
    padding: spacing.md,
  },
});
```

## Complete example

This example shows the component in a fuller composition with the surrounding pieces it usually needs.

```tsx
import { ProgressBar } from "@stylextras/ui/progress-bar";

function ProgressBarExample() {
  return (
    <>
      <div>
        <ProgressBar label="Build" max={100} value={20} />
        <ProgressBar label="Build" max={100} value={50} />
        <ProgressBar label="Build" max={100} value={90} />
      </div>
    </>
  );
}
```

## Accessibility notes

Make status text explicit. Use live regions only for updates that need to be announced immediately.

## Implementation notes

- Styles use token variables from `@stylextras/ui/tokens/*`.
- The implementation lives in `packages/ui/src/progress-bar/index.tsx`.
