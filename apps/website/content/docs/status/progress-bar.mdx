---
title: Progress Bar
description: Stylextras wrapper component.
---

Wraps `aria.ProgressBar` from `react-aria-components`, omitting `className` and `style` and adding `style?: StyleXStyles`.

## Demo
```tsx
import { ProgressBar } from '@stylextras/ui/progress-bar';

<ProgressBar />
```

## Props
```ts
import type { ProgressBarProps } from '@stylextras/ui/progress-bar';
```
