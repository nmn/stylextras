---
title: Toast
description: Stylextras wrapper component.
---

Wraps `aria.UNSTABLE_Toast` from `react-aria-components`, omitting `className` and `style` and adding `style?: StyleXStyles`.

## Demo
```tsx
import { Toast } from '@/components/stylextras';

<Toast />
```

## Props
```ts
import type { ToastProps } from '@/components/stylextras';
```
