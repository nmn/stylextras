# Alert Callout

Use Alert Callout to communicate state, progress, or feedback in a compact token-driven surface.

The implementation is intentionally small: visual defaults come from token variables, and the public props keep native HTML attributes available where the underlying element supports them.

## Import

```tsx
import { AlertCallout } from "@stylextras/ui/alert-callout";
```

## Basic usage

Start here when wiring the component into an interface. The demo shows the expected behavior, and the code block shows the smallest useful invocation.

<Suspense fallback={null}>
  <Example />
</Suspense>

```tsx
import { AlertCallout } from "@stylextras/ui/alert-callout";

<AlertCallout variant="warning">
  Review required fields before continuing.
</AlertCallout>;
```

## Props

The public prop types are exported from `@stylextras/ui/alert-callout`. Most components also accept an `sx` prop for custom styles and forward safe native HTML attributes for the underlying element.

```tsx
export type AlertProps = Omit<BaseProps, "className" | "style"> & {
  sx?: StyleXStyles;
  variant?: AlertVariant;
};
```

## Customize styles

Use the `sx` prop to compose custom styles with the component's token-driven defaults.

```tsx
import * as stylex from "@stylexjs/stylex";
import { colors } from "@stylextras/ui/tokens/color";
import { spacing } from "@stylextras/ui/tokens/spacing";
import { AlertCallout } from "@stylextras/ui/alert-callout";

<AlertCallout sx={styles.emphasis}>Example</AlertCallout>;

const styles = stylex.create({
  emphasis: {
    backgroundColor: colors.bgSubtle,
    padding: spacing.md,
  },
});
```

## Complete example

This example shows the component in a fuller composition with the surrounding pieces it usually needs.

```tsx
import { AlertCallout } from "@stylextras/ui/alert-callout";

function AlertCalloutExample() {
  return (
    <>
      <div>
        <AlertCallout variant="neutral">Neutral message</AlertCallout>
        <AlertCallout variant="info">Info message</AlertCallout>
        <AlertCallout variant="success">Success message</AlertCallout>
        <AlertCallout variant="warning">Warning message</AlertCallout>
        <AlertCallout variant="danger">Danger message</AlertCallout>
      </div>
    </>
  );
}
```

## Accessibility notes

Make status text explicit. Use live regions only for updates that need to be announced immediately.

## Implementation notes

- Styles use token variables from `@stylextras/ui/tokens/*`.
- The implementation lives in `packages/ui/src/alert-callout/index.tsx`.
