---
title: Meter
description: Stylextras wrapper component.
---

Wraps `aria.Meter` from `react-aria-components`, omitting `className` and `style` and adding `style?: StyleXStyles`.

## Demo
```tsx
import { Meter } from '@stylextras/ui/meter';

<Meter />
```

## Props
```ts
import type { MeterProps } from '@stylextras/ui/meter';
```
