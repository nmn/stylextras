`Badge` renders a native `span` with token-driven padding, radius, typography, and semantic color variants.

<BadgeExample />

## Notes

- Renders a native `span`
- Useful for status chips, labels, and small metadata
- Keeps the API minimal: size, variant, and `sx`

## Props

- `size?: 'sm' | 'md'`
- `variant?: 'neutral' | 'brand' | 'info' | 'success' | 'warning' | 'danger'`
- `sx?: StyleXStyles`
