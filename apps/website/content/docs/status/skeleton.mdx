---
title: Skeleton
description: Stylextras wrapper component.
---

Wraps `aria.Text` from `react-aria-components`, omitting `className` and `style` and adding `style?: StyleXStyles`.

## Demo
```tsx
import { Skeleton } from '@stylextras/ui/skeleton';

<Skeleton />
```

## Props
```ts
import type { SkeletonProps } from '@stylextras/ui/skeleton';
```
