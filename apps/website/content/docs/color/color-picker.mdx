---
title: Color Picker
description: Stylextras wrapper component.
---

Wraps `aria.ColorPicker` from `react-aria-components`, omitting `className` and `style` and adding `style?: StyleXStyles`.

## Demo
```tsx
import { ColorPicker } from '@stylextras/ui/color-picker';

<ColorPicker />
```

## Props
```ts
import type { ColorPickerProps } from '@stylextras/ui/color-picker';
```
