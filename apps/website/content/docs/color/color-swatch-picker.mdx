# Color Swatch Picker

Use Color Swatch Picker when a workflow needs color selection or color presentation without replacing native input semantics.

The implementation is intentionally small: visual defaults come from token variables, and the public props keep native HTML attributes available where the underlying element supports them.

## Import

```tsx
import { ColorSwatchPicker } from "@stylextras/ui/color-swatch-picker";
```

## Basic usage

Start here when wiring the component into an interface. The demo shows the expected behavior, and the code block shows the smallest useful invocation.

```tsx
import { ColorSwatchPicker } from "@stylextras/ui/color-swatch-picker";

<ColorSwatchPicker>Example</ColorSwatchPicker>;
```

## Props

The public prop types are exported from `@stylextras/ui/color-swatch-picker`. Most components also accept an `sx` prop for custom styles and forward safe native HTML attributes for the underlying element.

```tsx
export type ColorSwatchPickerProps = Omit<BaseProps, "className" | "style"> & {
  sx?: StyleXStyles;
  colors?: string[];
  value?: string;
  onValueChange?: (value: string) => void;
};
```

## Customize styles

Use the `sx` prop to compose custom styles with the component's token-driven defaults.

```tsx
import * as stylex from "@stylexjs/stylex";
import { colors } from "@stylextras/ui/tokens/color";
import { spacing } from "@stylextras/ui/tokens/spacing";
import { ColorSwatchPicker } from "@stylextras/ui/color-swatch-picker";

<ColorSwatchPicker aria-label="Example" sx={styles.emphasis} />;

const styles = stylex.create({
  emphasis: {
    backgroundColor: colors.bgSubtle,
    padding: spacing.md,
  },
});
```

## Complete example

This example shows the component in a fuller composition with the surrounding pieces it usually needs.

```tsx
import { useState } from "react";
import { ColorSwatchPicker } from "@stylextras/ui/color-swatch-picker";

function SwatchPickerDemo() {
  const [value, setValue] = useState("#2563eb");
  return (
    <div>
      <ColorSwatchPicker
        colors={["#2563eb", "#0f766e", "#9333ea", "#ea580c", "#dc2626"]}
        value={value}
        onValueChange={setValue}
      />
      <span>Selected: {value}</span>
    </div>
  );
}

function ColorSwatchPickerExample() {
  return (
    <>
      <SwatchPickerDemo />
    </>
  );
}
```

## Accessibility notes

Always provide an accessible name for color controls, and do not rely on color alone to communicate state.

## Implementation notes

- Styles use token variables from `@stylextras/ui/tokens/*`.
- The implementation lives in `packages/ui/src/color-swatch-picker/index.tsx`.
