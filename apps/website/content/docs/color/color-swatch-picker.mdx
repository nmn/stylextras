---
title: Color Swatch Picker
description: Stylextras wrapper component.
---

Wraps `aria.ColorSwatchPicker` from `react-aria-components`, omitting `className` and `style` and adding `style?: StyleXStyles`.

## Demo
```tsx
import { ColorSwatchPicker } from '@/components/stylextras';

<ColorSwatchPicker />
```

## Props
```ts
import type { ColorSwatchPickerProps } from '@/components/stylextras';
```
