# Color Swatch

Use Color Swatch when a workflow needs color selection or color presentation without replacing native input semantics.

The implementation is intentionally small: visual defaults come from token variables, and the public props keep native HTML attributes available where the underlying element supports them.

## Import

```tsx
import { ColorSwatch } from "@stylextras/ui/color-swatch";
```

## Basic usage

Start here when wiring the component into an interface. The demo shows the expected behavior, and the code block shows the smallest useful invocation.

<Suspense fallback={null}>
  <Example />
</Suspense>

```tsx
import { ColorSwatch } from "@stylextras/ui/color-swatch";

<ColorSwatch aria-label="Indigo" color="#4f46e5" />;
```

## Props

The public prop types are exported from `@stylextras/ui/color-swatch`. Most components also accept an `sx` prop for custom styles and forward safe native HTML attributes for the underlying element.

```tsx
export type ColorSwatchProps = Omit<BaseProps, "className" | "style"> & {
  sx?: StyleXStyles;
  color?: string;
};
```

## Customize styles

Use the `sx` prop to compose custom styles with the component's token-driven defaults.

```tsx
import * as stylex from "@stylexjs/stylex";
import { colors } from "@stylextras/ui/tokens/color";
import { spacing } from "@stylextras/ui/tokens/spacing";
import { ColorSwatch } from "@stylextras/ui/color-swatch";

<ColorSwatch aria-label="Example" sx={styles.emphasis} />;

const styles = stylex.create({
  emphasis: {
    backgroundColor: colors.bgSubtle,
    padding: spacing.md,
  },
});
```

## Complete example

This example shows the component in a fuller composition with the surrounding pieces it usually needs.

```tsx
import { ColorSwatch } from "@stylextras/ui/color-swatch";

function ColorSwatchExample() {
  return (
    <>
      <div>
        <ColorSwatch color="#2563eb" aria-label="Blue" />
        <ColorSwatch color="#9333ea" aria-label="Purple" />
        <ColorSwatch color="#db2777" aria-label="Pink" />
        <ColorSwatch color="#ea580c" aria-label="Orange" />
      </div>
    </>
  );
}
```

## Accessibility notes

Always provide an accessible name for color controls, and do not rely on color alone to communicate state.

## Implementation notes

- Styles use token variables from `@stylextras/ui/tokens/*`.
- The implementation lives in `packages/ui/src/color-swatch/index.tsx`.
