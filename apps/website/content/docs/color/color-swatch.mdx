---
title: Color Swatch
description: Stylextras wrapper component.
---

Wraps `aria.ColorSwatch` from `react-aria-components`, omitting `className` and `style` and adding `style?: StyleXStyles`.

## Demo
```tsx
import { ColorSwatch } from '@stylextras/ui/color-swatch';

<ColorSwatch />
```

## Props
```ts
import type { ColorSwatchProps } from '@stylextras/ui/color-swatch';
```
