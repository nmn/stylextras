---
title: Color Wheel
description: Stylextras wrapper component.
---

Wraps `aria.ColorWheel` from `react-aria-components`, omitting `className` and `style` and adding `style?: StyleXStyles`.

## Demo
```tsx
import { ColorWheel } from '@stylextras/ui/color-wheel';

<ColorWheel />
```

## Props
```ts
import type { ColorWheelProps } from '@stylextras/ui/color-wheel';
```
