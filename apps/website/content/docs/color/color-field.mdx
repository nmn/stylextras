---
title: Color Field
description: Stylextras wrapper component.
---

Wraps `aria.ColorField` from `react-aria-components`, omitting `className` and `style` and adding `style?: StyleXStyles`.

## Demo
```tsx
import { ColorField } from '@/components/stylextras';

<ColorField />
```

## Props
```ts
import type { ColorFieldProps } from '@/components/stylextras';
```
