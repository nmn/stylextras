---
title: Link
description: Stylextras wrapper component.
---

Wraps `aria.Link` from `react-aria-components`, omitting `className` and `style` and adding `style?: StyleXStyles`.

## Demo
```tsx
import { Link } from '@stylextras/ui/link';

<Link />
```

## Props
```ts
import type { LinkProps } from '@stylextras/ui/link';
```
