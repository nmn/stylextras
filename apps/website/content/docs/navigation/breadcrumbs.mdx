# Breadcrumbs

Use Breadcrumbs for navigation patterns that need predictable structure, token-driven spacing, and clear focus behavior.

The implementation is intentionally small: visual defaults come from token variables, and the public props keep native HTML attributes available where the underlying element supports them.

Search aliases: breadcrumb, path nav, trail navigation.

## Import

```tsx
import { Breadcrumbs } from "@stylextras/ui/breadcrumbs";
```

## Basic usage

Start here when wiring the component into an interface. The demo shows the expected behavior, and the code block shows the smallest useful invocation.

<Suspense fallback={null}>
  <Example />
</Suspense>

```tsx
import { Breadcrumbs } from "@stylextras/ui/breadcrumbs";

<Breadcrumbs items={[{ label: "Docs", href: "/docs" }, { label: "Button" }]} />;
```

## Props

The public prop types are exported from `@stylextras/ui/breadcrumbs`. Most components also accept an `sx` prop for custom styles and forward safe native HTML attributes for the underlying element.

```tsx
export type BreadcrumbsProps = BreadcrumbsProps;
```

## Complete example

This example shows the component in a fuller composition with the surrounding pieces it usually needs.

```tsx
import { Breadcrumbs } from "@stylextras/ui/breadcrumbs";

function BreadcrumbsExample() {
  return (
    <>
      <div>
        <Breadcrumbs
          items={[{ label: "Docs", href: "#docs" }, { label: "Buttons" }]}
        />
        <Breadcrumbs
          items={[
            { label: "Docs", href: "#docs" },
            { label: "Components", href: "#components" },
            { label: "Navigation", href: "#navigation" },
            { label: "Tabs" },
          ]}
        />
      </div>
    </>
  );
}
```

## Longer path

This pattern shows another common way to compose the component.

```tsx
import { Breadcrumbs } from "@stylextras/ui/breadcrumbs";

<Breadcrumbs
  items={[
    { label: "Docs", href: "/docs" },
    { label: "Components", href: "/docs/components" },
    { label: "Breadcrumbs" },
  ]}
/>;
```

## Accessibility notes

Use real links for navigation, mark the current location where appropriate, and keep focus order consistent with visual order.

## Implementation notes

- Styles use token variables from `@stylextras/ui/tokens/*`.
- The implementation lives in `packages/ui/src/breadcrumbs/index.tsx`.
