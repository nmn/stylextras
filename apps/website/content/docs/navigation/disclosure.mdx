---
title: Disclosure
description: Stylextras wrapper component.
---

Wraps `aria.Disclosure` from `react-aria-components`, omitting `className` and `style` and adding `style?: StyleXStyles`.

## Demo
```tsx
import { Disclosure } from '@stylextras/ui/disclosure';

<Disclosure />
```

## Props
```ts
import type { DisclosureProps } from '@stylextras/ui/disclosure';
```
