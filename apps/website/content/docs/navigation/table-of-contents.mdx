# Table Of Contents

Use Table Of Contents for navigation patterns that need predictable structure, token-driven spacing, and clear focus behavior.

The implementation is intentionally small: visual defaults come from token variables, and the public props keep native HTML attributes available where the underlying element supports them.

## Import

```tsx
import { TableOfContents } from "@stylextras/ui/table-of-contents";
```

## Basic usage

Start here when wiring the component into an interface. The demo shows the expected behavior, and the code block shows the smallest useful invocation.

<Suspense fallback={null}>
  <Example />
</Suspense>

```tsx
import { TableOfContents } from "@stylextras/ui/table-of-contents";

<TableOfContents
  items={[
    { label: "Props", href: "#props" },
    { label: "Examples", href: "#examples" },
  ]}
/>;
```

## Props

The public prop types are exported from `@stylextras/ui/table-of-contents`. Most components also accept an `sx` prop for custom styles and forward safe native HTML attributes for the underlying element.

```tsx
export type TableOfContentsProps = Omit<BaseProps, "className" | "style"> & {
  items: TableOfContentsItem[];
  sx?: StyleXStyles;
  title?: string;
};
```

## Customize styles

Use the `sx` prop to compose custom styles with the component's token-driven defaults.

```tsx
import * as stylex from "@stylexjs/stylex";
import { colors } from "@stylextras/ui/tokens/color";
import { spacing } from "@stylextras/ui/tokens/spacing";
import { TableOfContents } from "@stylextras/ui/table-of-contents";

<TableOfContents sx={styles.emphasis}>Example</TableOfContents>;

const styles = stylex.create({
  emphasis: {
    backgroundColor: colors.bgSubtle,
    padding: spacing.md,
  },
});
```

## Complete example

This example shows the component in a fuller composition with the surrounding pieces it usually needs.

```tsx
import { TableOfContents } from "@stylextras/ui/table-of-contents";

function TableOfContentsExample() {
  return (
    <>
      <TableOfContents
        title="On this page"
        items={[
          { label: "Overview", href: "#overview" },
          { label: "Usage", href: "#usage" },
          { label: "Examples", href: "#examples" },
          { label: "Accessibility", href: "#accessibility" },
        ]}
      />
    </>
  );
}
```

## Accessibility notes

Use real links for navigation, mark the current location where appropriate, and keep focus order consistent with visual order.

## Implementation notes

- Styles use token variables from `@stylextras/ui/tokens/*`.
- The implementation lives in `packages/ui/src/table-of-contents/index.tsx`.
