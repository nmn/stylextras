---
title: DisclosureGroup
description: Stylextras wrapper component.
---

Wraps `aria.DisclosureGroup` from `react-aria-components`, omitting `className` and `style` and adding `style?: StyleXStyles`.

## Demo
```tsx
import { DisclosureGroup } from '@stylextras/ui/disclosure-group';

<DisclosureGroup />
```

## Props
```ts
import type { DisclosureGroupProps } from '@stylextras/ui/disclosure-group';
```
