# Tabs

Use Tabs for navigation patterns that need predictable structure, token-driven spacing, and clear focus behavior.

The implementation is intentionally small: visual defaults come from token variables, and the public props keep native HTML attributes available where the underlying element supports them.

## Import

```tsx
import { Tabs } from "@stylextras/ui/tabs";
```

## Basic usage

Start here when wiring the component into an interface. The demo shows the expected behavior, and the code block shows the smallest useful invocation.

<Suspense fallback={null}>
  <Example />
</Suspense>

```tsx
import { Tabs } from "@stylextras/ui/tabs";

<Tabs>Example</Tabs>;
```

## Props

The public prop types are exported from `@stylextras/ui/tabs`. Most components also accept an `sx` prop for custom styles and forward safe native HTML attributes for the underlying element.

```tsx
export type TabsProps = Omit<BaseProps, "className" | "style"> & {
  sx?: StyleXStyles;
  tabs?: TabItem[];
  value?: string;
  defaultValue?: string;
  onValueChange?: (value: string) => void;
};
```

## Customize styles

Use the `sx` prop to compose custom styles with the component's token-driven defaults.

```tsx
import * as stylex from "@stylexjs/stylex";
import { colors } from "@stylextras/ui/tokens/color";
import { spacing } from "@stylextras/ui/tokens/spacing";
import { Tabs } from "@stylextras/ui/tabs";

<Tabs sx={styles.emphasis}>Example</Tabs>;

const styles = stylex.create({
  emphasis: {
    backgroundColor: colors.bgSubtle,
    padding: spacing.md,
  },
});
```

## Complete example

This example shows the component in a fuller composition with the surrounding pieces it usually needs.

```tsx
import { Tabs } from "@stylextras/ui/tabs";

function TabsExample() {
  return (
    <>
      <div>
        <Tabs
          tabs={["Overview", "Examples", "Accessibility"]}
          defaultValue="Overview"
        />
        <Tabs
          tabs={["General", "Tokens", "Publishing"]}
          defaultValue="Tokens"
        />
      </div>
    </>
  );
}
```

## Accessibility notes

Use real links for navigation, mark the current location where appropriate, and keep focus order consistent with visual order.

## Implementation notes

- Styles use token variables from `@stylextras/ui/tokens/*`.
- The implementation lives in `packages/ui/src/tabs/index.tsx`.
