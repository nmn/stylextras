# Pagination

Use Pagination for navigation patterns that need predictable structure, token-driven spacing, and clear focus behavior.

The implementation is intentionally small: visual defaults come from token variables, and the public props keep native HTML attributes available where the underlying element supports them.

## Import

```tsx
import { Pagination } from "@stylextras/ui/pagination";
```

## Basic usage

Start here when wiring the component into an interface. The demo shows the expected behavior, and the code block shows the smallest useful invocation.

<Suspense fallback={null}>
  <Example />
</Suspense>

```tsx
import { Pagination } from "@stylextras/ui/pagination";

<Pagination>Example</Pagination>;
```

## Props

The public prop types are exported from `@stylextras/ui/pagination`. Most components also accept an `sx` prop for custom styles and forward safe native HTML attributes for the underlying element.

```tsx
export type PaginationProps = Omit<BaseProps, "className" | "style"> & {
  sx?: StyleXStyles;
  currentPage?: number;
  totalPages?: number;
};
```

## Customize styles

Use the `sx` prop to compose custom styles with the component's token-driven defaults.

```tsx
import * as stylex from "@stylexjs/stylex";
import { colors } from "@stylextras/ui/tokens/color";
import { spacing } from "@stylextras/ui/tokens/spacing";
import { Pagination } from "@stylextras/ui/pagination";

<Pagination sx={styles.emphasis}>Example</Pagination>;

const styles = stylex.create({
  emphasis: {
    backgroundColor: colors.bgSubtle,
    padding: spacing.md,
  },
});
```

## Complete example

This example shows the component in a fuller composition with the surrounding pieces it usually needs.

```tsx
import { Pagination } from "@stylextras/ui/pagination";

function PaginationExample() {
  return (
    <>
      <div>
        <Pagination currentPage={2} totalPages={5} />
        <Pagination currentPage={4} totalPages={8} />
      </div>
    </>
  );
}
```

## Accessibility notes

Use real links for navigation, mark the current location where appropriate, and keep focus order consistent with visual order.

## Implementation notes

- Styles use token variables from `@stylextras/ui/tokens/*`.
- The implementation lives in `packages/ui/src/pagination/index.tsx`.
