---
title: Breadcrumb
description: Stylextras wrapper component.
---

Wraps `aria.Breadcrumbs` from `react-aria-components`, omitting `className` and `style` and adding `style?: StyleXStyles`.

## Demo
```tsx
import { Breadcrumb } from '@stylextras/ui/breadcrumb';

<Breadcrumb />
```

## Props
```ts
import type { BreadcrumbProps } from '@stylextras/ui/breadcrumb';
```
