# Breadcrumb

Use Breadcrumb for navigation patterns that need predictable structure, token-driven spacing, and clear focus behavior.

The implementation is intentionally small: visual defaults come from token variables, and the public props keep native HTML attributes available where the underlying element supports them.

## Import

```tsx
import { Breadcrumb } from "@stylextras/ui/breadcrumb";
```

## Basic usage

Start here when wiring the component into an interface. The demo shows the expected behavior, and the code block shows the smallest useful invocation.

<Suspense fallback={null}>
  <Example />
</Suspense>

```tsx
import { Breadcrumb } from "@stylextras/ui/breadcrumb";

<Breadcrumb items={[{ label: "Docs", href: "/docs" }, { label: "Button" }]} />;
```

## Props

The public prop types are exported from `@stylextras/ui/breadcrumb`. Most components also accept an `sx` prop for custom styles and forward safe native HTML attributes for the underlying element.

```tsx
export type BreadcrumbProps = Omit<BaseProps, "className" | "style"> & {
  sx?: StyleXStyles;
  items?: BreadcrumbItem[];
};
```

## Customize styles

Use the `sx` prop to compose custom styles with the component's token-driven defaults.

```tsx
import * as stylex from "@stylexjs/stylex";
import { colors } from "@stylextras/ui/tokens/color";
import { spacing } from "@stylextras/ui/tokens/spacing";
import { Breadcrumb } from "@stylextras/ui/breadcrumb";

<Breadcrumb items={[{ label: "Docs", href: "/docs" }]} sx={styles.emphasis} />;

const styles = stylex.create({
  emphasis: {
    backgroundColor: colors.bgSubtle,
    padding: spacing.md,
  },
});
```

## Complete example

This example shows the component in a fuller composition with the surrounding pieces it usually needs.

```tsx
import { Breadcrumb } from "@stylextras/ui/breadcrumb";

function BreadcrumbExample() {
  return (
    <>
      <div>
        <Breadcrumb
          items={[{ label: "Docs", href: "#docs" }, { label: "Buttons" }]}
        />
        <Breadcrumb
          items={[
            { label: "Docs", href: "#docs" },
            { label: "Components", href: "#components" },
            { label: "Navigation", href: "#navigation" },
            { label: "Tabs" },
          ]}
        />
      </div>
    </>
  );
}
```

## Accessibility notes

Use real links for navigation, mark the current location where appropriate, and keep focus order consistent with visual order.

## Implementation notes

- Styles use token variables from `@stylextras/ui/tokens/*`.
- The implementation lives in `packages/ui/src/breadcrumb/index.tsx`.
