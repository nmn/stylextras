Stylextras is a set of headless-friendly React Aria wrappers and a token-driven
UI foundation built for StyleX.

## Install

```sh
bun add @stylextras/ui
```

## Import components

Import each component from its own path.

```tsx
import { Button } from '@stylextras/ui/button';

export function Example() {
  return <Button>Press me</Button>;
}
```

## Import tokens

The token system is split into focused sets so themes can override a small
number of core values and derive the rest.

```tsx
import { colors } from '@stylextras/ui/tokens/color';
import { spacing } from '@stylextras/ui/tokens/spacing';
```
