## Documentation website theme

Every variable group exports a `docs` theme that mirrors this site’s purple
primary and selection colors, neutral surfaces, and periwinkle-blue highlights
for hovered or keyboard-focused options and menu items, plus its system
typography, spacing, radii, borders, shadows, blur, and motion.

```tsx
import * as stylex from "@stylexjs/stylex";
import { blurThemes } from "@stylextras/ui/blur-themes";
import { colorThemes } from "@stylextras/ui/color-themes";
import { elevationThemes } from "@stylextras/ui/elevation-themes";
import { motionThemes } from "@stylextras/ui/motion-themes";
import { radiusThemes } from "@stylextras/ui/radius-themes";
import { spacingThemes } from "@stylextras/ui/spacing-themes";
import { strokeThemes } from "@stylextras/ui/stroke-themes";
import { typographyThemes } from "@stylextras/ui/typography-themes";

export function DocumentationTheme({ children }) {
  return (
    <section
      {...stylex.props(
        colorThemes.docs,
        spacingThemes.docs,
        radiusThemes.docs,
        strokeThemes.docs,
        typographyThemes.docs,
        elevationThemes.docs,
        blurThemes.docs,
        motionThemes.docs,
      )}
    >
      {children}
    </section>
  );
}
```

## Composition

Color, spacing, radius, stroke, typography, elevation, blur, and motion remain
independent variable groups. A theme overrides core values in one group; its
same-set derived values recompute automatically.
