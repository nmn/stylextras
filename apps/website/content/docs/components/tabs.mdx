## Anatomy

- `Tabs` — Styled `<div>`.
- `TabsContent` — Styled `<div>`.
- `TabsList` — Styled `<div>`.
- `TabsTrigger` — Styled `<button>`.

## Accessibility and browser behavior

Each part renders and styles the native element named above. The API reference states the exact React or inherited prop base, including whether `ref` is accepted, every omitted native prop, and every component-applied default. Any native prop not named in `Omit` remains accepted, including standard ARIA/data attributes and event handlers. `sx` is composed last.

This component uses a focused client controller for composite state and keyboard interaction while retaining native elements for focus and form semantics where possible. When the relevant browser interaction API is unavailable, the package loads a narrowly scoped, feature-detected bridge for that behavior.

## API reference

### Tabs

`TabsProps`

Complete native/inherited prop base: `Omit<React.ComponentPropsWithRef<"div">, "className" | "defaultValue" | "style">`. The props below complete `TabsProps`.

- `activationMode` — `"manual" | "automatic"`. Default: `'automatic'`.
- `defaultValue` — `string`. Default: `''`.
- `onValueChange` — `(value: string) => void`.
- `orientation` — `"vertical" | "horizontal"`. Default: `'horizontal'`.
- `sx` — `stylex.StyleXStyles`.
- `value` — `string`.

### TabsContent

`TabsContentProps`

Complete native/inherited prop base: `Omit<React.ComponentPropsWithRef<"div">, "aria-labelledby" | "className" | "role" | "style">`. The props below complete `TabsContentProps`.

- `sx` — `stylex.StyleXStyles`.
- `tabIndex` — `TabsContentProps["tabIndex"]`. Default: `0`.
- `value` — `string`. Required.

### TabsList

`TabsListProps`

Complete native/inherited prop base: `Omit<React.ComponentPropsWithRef<"div">, "className" | "role" | "style">`. The props below complete `TabsListProps`.

- `sx` — `stylex.StyleXStyles`.

### TabsTrigger

`TabsTriggerProps`

Complete native/inherited prop base: `Omit<React.ComponentPropsWithRef<"button">, "aria-controls" | "aria-selected" | "className" | "role" | "style" | "value">`. The props below complete `TabsTriggerProps`.

- `sx` — `stylex.StyleXStyles`.
- `type` — `TabsTriggerProps["type"]`. Default: `'button'`.
- `value` — `string`. Required.
