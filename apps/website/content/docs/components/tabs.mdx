## Anatomy

- `Tabs` — Styled `<div>`.
- `TabsContent` — Styled `<div>`.
- `TabsList` — Styled `<div>`.
- `TabsTrigger` — Styled `<button>`.

## Accessibility and browser behavior

Each part renders and styles the native element named above. The API reference states the exact React or inherited prop base, including whether `ref` is accepted, every omitted native prop, and every component-applied default. Any native prop not named in `Omit` remains accepted, including standard ARIA/data attributes and event handlers. `sx` is composed last.

This component implements the tab keyboard pattern directly, including RTL horizontal arrows, without loading the focusgroup bridge. Every tablist requires an accessible name. Generated IDs, relationships, roles, orientation, selection, visibility, and tab stops are component-owned and cannot be overridden by consumer props. If the selected value points to a disabled tab, the first enabled tab becomes the roving tab stop. Use manual activation when panel content is deferred or otherwise not instantly available.

## API reference

### Tabs

`TabsProps`

Complete native/inherited prop base: `Omit<React.ComponentPropsWithRef<"div">, "className" | "defaultValue" | "style">`. The props below complete `TabsProps`.

- `activationMode` — `"manual" | "automatic"`. Default: `'automatic'`.
- `defaultValue` — `string`. Default: `''`.
- `onValueChange` — `(value: string) => void`.
- `orientation` — `"vertical" | "horizontal"`. Default: `'horizontal'`.
- `sx` — `stylex.StyleXStyles`.
- `value` — `string`.

### TabsContent

`TabsContentProps`

Complete native/inherited prop base: `Omit<React.ComponentPropsWithRef<"div">, "aria-labelledby" | "className" | "hidden" | "id" | "role" | "style" | "tabIndex">`. The props below complete `TabsContentProps`.

- `sx` — `stylex.StyleXStyles`.
- `value` — `string`. Required.

### TabsList

`TabsListProps`

Complete native/inherited prop base: native div props with component-owned `role`, `aria-orientation`, and styles. `aria-label` or `aria-labelledby` is required.

- `sx` — `stylex.StyleXStyles`.

### TabsTrigger

`TabsTriggerProps`

Complete native/inherited prop base: `Omit<React.ComponentPropsWithRef<"button">, "aria-controls" | "aria-selected" | "className" | "id" | "role" | "style" | "tabIndex" | "type" | "value">`. The props below complete `TabsTriggerProps`.

- `sx` — `stylex.StyleXStyles`.
- `value` — `string`. Required.
