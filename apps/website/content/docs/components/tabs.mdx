---
title: Tabs
description: Stylextras wrapper component.
---

Wraps `aria.Tabs` from `react-aria-components`, omitting `className` and `style` and adding `style?: StyleXStyles`.

## Demo
```tsx
import { Tabs } from '@/components/stylextras';

<Tabs />
```

## Props
```ts
import type { TabsProps } from '@/components/stylextras';
```
