---
title: Timefield
description: Stylextras wrapper component.
---

Wraps `aria.TimeField` from `react-aria-components`, omitting `className` and `style` and adding `style?: StyleXStyles`.

## Demo
```tsx
import { TimeField } from '@/components/stylextras';

<TimeField />
```

## Props
```ts
import type { TimeFieldProps } from '@/components/stylextras';
```
