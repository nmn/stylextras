---
title: Color Area
description: Stylextras wrapper component.
---

Wraps `aria.ColorArea` from `react-aria-components`, omitting `className` and `style` and adding `style?: StyleXStyles`.

## Demo
```tsx
import { ColorArea } from '@/components/stylextras';

<ColorArea />
```

## Props
```ts
import type { ColorAreaProps } from '@/components/stylextras';
```
