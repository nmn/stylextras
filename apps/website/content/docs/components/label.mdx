---
title: Label
description: Stylextras wrapper component.
---

Wraps `aria.Label` from `react-aria-components`, omitting `className` and `style` and adding `style?: StyleXStyles`.

## Demo
```tsx
import { Label } from '@/components/stylextras';

<Label />
```

## Props
```ts
import type { LabelProps } from '@/components/stylextras';
```
