### Example code

```tsx
'use client';

import * as stylex from '@stylexjs/stylex';
import { colors } from '../tokens/color.stylex';
import { spacing } from '../tokens/spacing.stylex';
import { Resizable, ResizableHandle, ResizablePanel } from './index';

export default function Example() {
  return (
    <Resizable defaultValue={38} sx={styles.root}>
      <ResizablePanel sx={styles.panel}>Navigation and files</ResizablePanel>
      <ResizableHandle label="Resize editor panels" />
      <ResizablePanel sx={styles.panel}>Editor preview</ResizablePanel>
    </Resizable>
  );
}

const styles = stylex.create({
  root: {
    borderColor: colors.border,
    borderStyle: 'solid',
    borderWidth: 1,
    height: '13rem',
    width: '100%',
  },
  panel: {
    backgroundColor: colors.surface,
    color: colors.fgMuted,
    padding: spacing.md,
  },
});
```

## Accessibility and browser behavior

Each part renders and styles the native element named above. The API reference states the exact React or inherited prop base, including whether `ref` is accepted, every omitted native prop, and every component-applied default. Any native prop not named in `Omit` remains accepted, including standard ARIA/data attributes and event handlers. `sx` is composed last.

This component uses a focused client controller for composite state and keyboard interaction while retaining native elements for focus and form semantics where possible. Its behavior comes from the component controller rather than a browser-feature polyfill.

## API reference

### Resizable

`ResizableProps`

Complete native/inherited prop base: `Omit<React.ComponentPropsWithRef<"div">, "className" | "defaultValue" | "style">`. The props below complete `ResizableProps`.

- `defaultValue` — `number`. Default: `50`.
- `direction` — `"vertical" | "horizontal"`. Default: `'horizontal'`.
- `max` — `number`. Default: `90`.
- `min` — `number`. Default: `10`.
- `onValueChange` — `(value: number) => void`.
- `sx` — `stylex.StyleXStyles`.
- `value` — `number`.

### ResizableHandle

`ResizableHandleProps`

Complete native/inherited prop base: `Omit<React.ComponentPropsWithRef<"div">, "aria-orientation" | "aria-valuemax" | "aria-valuemin" | "aria-valuenow" | "className" | "role" | "style">`. The props below complete `ResizableHandleProps`.

- `label` — `string`. Required.
- `sx` — `stylex.StyleXStyles`.
- `tabIndex` — `ResizableHandleProps["tabIndex"]`. Default: `0`.

### ResizablePanel

`ResizablePanelProps`

Complete native/inherited prop base: `Omit<React.ComponentPropsWithRef<"div">, "className" | "style">`. The props below complete `ResizablePanelProps`.

- `sx` — `stylex.StyleXStyles`.
