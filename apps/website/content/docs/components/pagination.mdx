---
title: Pagination
description: Stylextras wrapper component.
---

Wraps `aria.ListBoxLoadMoreItem` from `react-aria-components`, omitting `className` and `style` and adding `style?: StyleXStyles`.

## Demo
```tsx
import { Pagination } from '@/components/stylextras';

<Pagination />
```

## Props
```ts
import type { PaginationProps } from '@/components/stylextras';
```
