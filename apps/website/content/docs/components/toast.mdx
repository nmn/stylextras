### Example code

```tsx
'use client';

import * as stylex from '@stylexjs/stylex';
import { Button } from '../button';
import { Toast, Toaster, ToastDescription, ToastTitle, toast } from './index';

export default function Example() {
  return (
    <>
      <ol aria-label="Toast variants" {...stylex.props(styles.list)}>
        <Toast>
          <div>
            <ToastTitle>Draft restored</ToastTitle>
            <ToastDescription>
              Your local changes are available again.
            </ToastDescription>
          </div>
        </Toast>
        <Toast variant="success">
          <div>
            <ToastTitle>Theme saved</ToastTitle>
            <ToastDescription>The Zinc preset is now active.</ToastDescription>
          </div>
        </Toast>
        <Toast variant="danger">
          <div>
            <ToastTitle>Publish failed</ToastTitle>
            <ToastDescription>
              Check the connection and try again.
            </ToastDescription>
          </div>
        </Toast>
      </ol>
      <Button
        onClick={() =>
          toast({
            title: 'Export complete',
            description: 'stylextras-ui-0.2.0-beta.tgz is ready.',
            duration: 8_000,
            variant: 'success',
          })
        }
      >
        Queue a toast
      </Button>

      <Toaster />
    </>
  );
}

const styles = stylex.create({
  list: {
    listStyle: 'none',
    margin: 0,
    padding: 0,
  },
});
```

## Anatomy

- `Toast` — Styled `<li>`.
- `ToastDescription` — Styled `<p>`.
- `Toaster` — Styled `<ol>`.
- `ToastTitle` — Styled `<div>`.

## Accessibility and browser behavior

Each part renders and styles the native element named above. The API reference states the exact React or inherited prop base, including whether `ref` is accepted, every omitted native prop, and every component-applied default. Any native prop not named in `Omit` remains accepted, including standard ARIA/data attributes and event handlers. `sx` is composed last.

The queue uses one polite and one assertive text-only announcer, keeping interactive controls outside live regions. Primitive string/number titles and descriptions are extracted safely; pass `announcement` whenever the visual content contains React elements. Batched records are announced together without cloning visible nodes, IDs, or controls. Visual urgency is independent from announcement priority. Timers pause while the page is hidden and while users hover or focus notifications; actionable notifications remain until dismissed unless a duration is supplied.

## API reference

### Toast

`ToastProps`

Complete native/inherited prop base: `Omit<React.ComponentPropsWithRef<"li">, "className" | "role" | "style">`. The props below complete `ToastProps`.

- `sx` — `stylex.StyleXStyles`.
- `priority` — `"polite" | "assertive" | "off"`. Default: `'polite'`.
- `variant` — `"default" | "success" | "danger"`. Default: `'default'`.

### toast

`toast(options)` queues a visual toast and returns `{ id, dismiss }`.

- `announcement` — `string`. Plain live-region text for rich `title` or `description` content.
- `description` — `ReactNode`.
- `duration` — `number`. Defaults to `4000`, or `0` when an action is present.
- `id` — `string`.
- `priority` — `"polite" | "assertive"`. Default: `'polite'`.
- `title` — `ReactNode`. Required.
- `variant` — `"default" | "success" | "danger"`. Default: `'default'`.

### ToastDescription

`ToastDescriptionProps`

Complete native/inherited prop base: `Omit<React.ComponentPropsWithRef<"p">, "className" | "style">`. The props below complete `ToastDescriptionProps`.

- `sx` — `stylex.StyleXStyles`.

### Toaster

`ToasterProps`

Complete native/inherited prop base: `Omit<React.ComponentPropsWithRef<"ol">, "children" | "className" | "style">`. The props below complete `ToasterProps`.

- `aria-label` — `ToasterProps["aria-label"]`. Default: `'Notifications'`.
- `dismissLabel` — `string`. Default: `'Dismiss notification'`.
- `limit` — `number`. Default: `3`.
- `sx` — `stylex.StyleXStyles`.

### ToastTitle

Complete native/inherited prop base: `Omit<React.ComponentPropsWithRef<"div">, "className" | "style">`. The props below complete `ToastTitleProps`.

- `sx` — `stylex.StyleXStyles`.
