---
title: Toggle
description: Stylextras wrapper component.
---

Wraps `aria.ToggleButton` from `react-aria-components`, omitting `className` and `style` and adding `style?: StyleXStyles`.

## Demo
```tsx
import { Toggle } from '@/components/stylextras';

<Toggle />
```

## Props
```ts
import type { ToggleProps } from '@/components/stylextras';
```
