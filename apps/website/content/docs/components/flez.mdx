---
title: Flez
description: Stylextras wrapper component.
---

Wraps `aria.Group` from `react-aria-components`, omitting `className` and `style` and adding `style?: StyleXStyles`.

## Demo
```tsx
import { Flez } from '@/components/stylextras';

<Flez />
```

## Props
```ts
import type { FlezProps } from '@/components/stylextras';
```
