---
title: File Drop zone
description: Stylextras wrapper component.
---

Wraps `aria.DropZone` from `react-aria-components`, omitting `className` and `style` and adding `style?: StyleXStyles`.

## Demo
```tsx
import { FileDropZone } from '@/components/stylextras';

<FileDropZone />
```

## Props
```ts
import type { FileDropZoneProps } from '@/components/stylextras';
```
