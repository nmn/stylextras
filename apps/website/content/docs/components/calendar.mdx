## Anatomy

- `Calendar` — Root `<div>` with month-navigation buttons and an ARIA grid.
- Each date — `role="gridcell"` wrapper containing a native `<button type="button">`.

## Accessibility and browser behavior

Each part renders and styles the native element named above. The API reference states the exact React or inherited prop base, including whether `ref` is accepted, every omitted native prop, and every component-applied default. Any native prop not named in `Omit` remains accepted, including standard ARIA/data attributes and event handlers. `sx` is composed last.

This component uses a focused client controller for composite state and keyboard interaction while retaining native elements for focus and form semantics where possible. Its behavior comes from the component controller rather than a browser-feature polyfill.

Every date button is named with its full localized date. Arrow keys respect text direction; Page Up/Down preserves the day where possible, and unavailable month buttons are disabled at `min`/`max` boundaries. Month-button navigation moves and clamps the grid's active date as well as the visible month, so exactly one enabled date remains in the roving tab sequence.

Pass `today` as an ISO `YYYY-MM-DD` value when the server already knows the user's local date. Without a selected value or `today`, the component renders its localized loading status consistently on the server and resolves the browser-local date after hydration, avoiding timezone-dependent hydration mismatches.

## API reference

### Calendar

`CalendarProps`

Complete native/inherited prop base: `Omit<React.ComponentPropsWithRef<"div">, "className" | "style">`. The props below complete `CalendarProps`.

- `defaultValue` — `string | (readonly string[] & string)`. Default: `''`.
- `loadingLabel` — `string`. Default: `'Loading calendar'`.
- `locale` — `string`. Default: `'en-US'`.
- `max` — `string`.
- `min` — `string`.
- `nextMonthLabel` — `string`. Default: `'Next month'`.
- `onValueChange` — `(value: string) => void`.
- `previousMonthLabel` — `string`. Default: `'Previous month'`.
- `sx` — `stylex.StyleXStyles`.
- `today` — `string`. Stable local date in `YYYY-MM-DD` form for server rendering and `aria-current="date"`.
- `value` — `string`.
- `weekStartsOn` — `number`. Default: `getWeekStartsOn(locale)`.
