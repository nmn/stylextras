---
title: Color Slider
description: Stylextras wrapper component.
---

Wraps `aria.ColorSlider` from `react-aria-components`, omitting `className` and `style` and adding `style?: StyleXStyles`.

## Demo
```tsx
import { ColorSlider } from '@/components/stylextras';

<ColorSlider />
```

## Props
```ts
import type { ColorSliderProps } from '@/components/stylextras';
```
