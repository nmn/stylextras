---
title: Table
description: Stylextras wrapper component.
---

Wraps `aria.Table` from `react-aria-components`, omitting `className` and `style` and adding `style?: StyleXStyles`.

## Demo
```tsx
import { Table } from '@/components/stylextras';

<Table />
```

## Props
```ts
import type { TableProps } from '@/components/stylextras';
```
