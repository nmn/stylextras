---
title: Form
description: Stylextras wrapper component.
---

Wraps `aria.Form` from `react-aria-components`, omitting `className` and `style` and adding `style?: StyleXStyles`.

## Demo
```tsx
import { Form } from '@/components/stylextras';

<Form />
```

## Props
```ts
import type { FormProps } from '@/components/stylextras';
```
