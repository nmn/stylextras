---
title: Kbd
description: Stylextras wrapper component.
---

Wraps `aria.Keyboard` from `react-aria-components`, omitting `className` and `style` and adding `style?: StyleXStyles`.

## Demo
```tsx
import { Kbd } from '@/components/stylextras';

<Kbd />
```

## Props
```ts
import type { KbdProps } from '@/components/stylextras';
```
